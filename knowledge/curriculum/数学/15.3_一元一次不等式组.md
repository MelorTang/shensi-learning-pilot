# 一元一次不等式组

## Core Idea
一元一次不等式组由几个含有同一个未知数的一元一次不等式组合而成。它的解集是这几个不等式解集的公共部分（交集）。寻找公共部分的口诀是：“同大取大，同小取小，大小小大中间找，大大小小解不了（无解）”。

## Learning Steps
1. **理解不等式组的概念**：认识到必须是求同一个未知数在多个条件下的共同范围。
2. **分别求出各个不等式的解集**：这是基础，需要熟练掌握单体一元一次不等式的解法。
3. **利用数轴求公共部分**：把所有解集画在同一个数轴上，找出重叠的部分。
4. **掌握口诀**：熟悉四种常见不等式组解集情况的口诀，提高做题速度。

## Common Pitfalls
- **数轴画法混乱**：多条线在数轴上不区分高度，导致看不清重叠部分。
- **端点包含与否的判断**：在交集处，一个包含端点一个不包含时，判断交集是否包含端点容易出错（实心和空心的交集是空心）。
- **不等式解错导致全盘皆输**：只要其中一个不等式解错，整个不等式组的解集大概率是错误的。

## Parent Guidance
引导孩子养成“必画数轴”的好习惯，绝不能只凭脑海想象或死记口诀来求交集。家长可以购买带网格的草稿纸，要求孩子画出清晰、规范的数轴，并且不同不等式的线要画在不同的高度。检查作业时，重点核对“公共部分”的判断是否准确。

## Linked Mistake Concepts
- 集合的交集概念（虽然初中不强调集合，但思想是一致的）
- 一元一次不等式（不等式组的基石，单体不会，组就不会）
- 逻辑“与”和“或”的区别（不等式组是“同时满足”，即“与”的关系）

## Examples
题目：解不等式组：
(1) 2x - 1 > x + 1
(2) \frac{1}{2}x + 3 \ge 2
```text
解：
解不等式(1)：
2x - x > 1 + 1
x > 2

解不等式(2)：
1/2 x ≥ -1
x ≥ -2

将(1)和(2)的解集在同一个数轴上表示出来：
x > 2 是在2处空心向右。
x ≥ -2 是在-2处实心向右。
两者的公共部分是 x > 2。
(根据口诀：同大取大，所以取 x > 2)
因此，原不等式组的解集为 x > 2。
```
