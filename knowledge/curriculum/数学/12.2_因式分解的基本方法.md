# 因式分解的基本方法

## Core Idea
因式分解的两种最基本方法是提公因式法和运用公式法。
1. 提公因式法：如果多项式的各项含有公因式，可以把这个公因式提出来，从而将多项式化成两个因式乘积的形式。
   公因式的确定法则：系数取各项系数的最大公约数，字母取各项相同字母的最低次幂。
2. 运用公式法：逆用乘法公式进行因式分解。
   - 平方差公式：$a^2 - b^2 = (a+b)(a-b)$
   - 完全平方公式：$a^2 \pm 2ab + b^2 = (a \pm b)^2$

## Learning Steps
1. 找公因式：第一步看系数的最大公约数，第二步找相同的字母，第三步确定相同字母的最低指数。
2. 提公因式：将公因式写在括号外面，括号里面是原多项式各项除以公因式的结果。
3. 识别公式特征：
   - 见到两项差，且都是平方的形式，尝试平方差公式。
   - 见到三项式，首尾均为完全平方且符号相同，中间项是首尾平方根乘积的2倍（或-2倍），尝试完全平方公式。
4. 综合首选法则：遇到多项式因式分解，**“首项有负常提负，各项有公先提公”**，提完公因式后再看括号内是否能用公式法。

## Common Pitfalls
- 漏掉“1”：提取公因式时，当多项式中某一项恰好是公因式本身，提取后这一项位置应留“$1$”，学生常错写成“$0$”或直接漏掉。
- 提负号未变号：首项系数为负时，通常提取负号，此时括号内的各项都必须改变符号，学生极易忘记变号。
- 公因式提取不彻底：如 $4x^3 - 8x^2$，只提取了 $2x^2$ 得到 $2x^2(2x - 4)$，括号内还可以再提 $2$。
- 假完全平方式混淆：看到 $a^2 + ab + b^2$，因为缺少系数 $2$，误以为可以用完全平方公式。

## Parent Guidance
- 步骤拆解法：要求孩子在提取公因式时分三步念出来：“第一找数字最大公约，第二找字母全都有的，第三找字母指数最小的”。
- 公式特征记忆：教孩子像认脸一样认完全平方式的“脸”——“头是方，尾是方，中间是头尾乘积的两倍”。可以用 $x^2 + 6x + 9$ 为例：头是 $x$ 的平方，尾是 $3$ 的平方，中间 $6x$ 正好是 $2 \times x \times 3$。
- 自检习惯：强制要求做完因式分解后，在草稿纸上用整式乘法乘回去，看看能不能变回原题。

## Linked Mistake Concepts
- 幂的运算法则混用：在提取公因式后写括号内剩余项时，对字母的指数处理，实际上是做同底数幂除法（指数相减），有些学生会做成减法。
- 平方差公式中符号处理：遇到 $-a^2 + b^2$ 时，不知所措。其实交换一下位置变成 $b^2 - a^2$ 即可看出平方差。

## Examples
```text
Question:
把下列各式分解因式：
(1) -12x^2 y + 18xy^2 - 6xy
(2) 4a^2 - 9b^2
(3) 2x^2 - 8x + 8

Solution:
(1) -12x^2 y + 18xy^2 - 6xy
  // 首项为负，提出 -6xy
  = -6xy (2x - 3y + 1)
  // 注意：提取负号后，括号内各项全部变号；最后面的 -6xy 提取后余下 +1。

(2) 4a^2 - 9b^2
  // 观察是两项的差，构造平方差的形式
  = (2a)^2 - (3b)^2
  = (2a + 3b)(2a - 3b)

(3) 2x^2 - 8x + 8
  // 各项都有公因数 2，必须先提公因式
  = 2(x^2 - 4x + 4)
  // 括号内是三项式，头是 x^2，尾是 2^2，中间是 -2*x*2，符合完全平方公式
  = 2(x - 2)^2
```
