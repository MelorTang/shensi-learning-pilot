# 乘法公式

## Core Idea
乘法公式是指多项式乘法中的两种特殊形式，它们是极其重要且常用的代数工具。
1. 平方差公式：$(a + b)(a - b) = a^2 - b^2$。两个数的和与这两个数的差的积，等于这两个数的平方差。
2. 完全平方公式：
   - $(a + b)^2 = a^2 + 2ab + b^2$ （两数和的完全平方）
   - $(a - b)^2 = a^2 - 2ab + b^2$ （两数差的完全平方）
两数和（或差）的平方，等于它们的平方和，加上（或减去）它们乘积的 2 倍。

## Learning Steps
1. 几何直观验证：通过面积模型直观理解公式。如将边长为 $(a+b)$ 的正方形面积分割为 $a^2, b^2$ 以及两个 $ab$ 的长方形面积之和。
2. 对号入座找 $a$ 和 $b$：应用公式时，核心是认清原式中的哪部分是公式中的 $a$，哪部分是公式中的 $b$。特别是当项包含系数和负号时。
3. 正向展开公式：熟练写出展开后的多项式，注意 $2ab$ 这一交叉项不可遗漏。
4. 符号变换与扩展：如 $(-a-b)^2 = (a+b)^2$，以及三个项的平方公式 $(a+b+c)^2 = a^2+b^2+c^2+2ab+2bc+2ca$ 的延伸。

## Common Pitfalls
- 完全平方漏掉中间项：将 $(a+b)^2$ 错写为 $a^2+b^2$。这是数学中最“顽固”的错误。
- 符号错乱：在平方差公式 $(a+b)(-a+b)$ 中，找错“符号相同的项（作为$a$）”和“符号相反的项（作为$b$）”。
- 项的系数未平方：如果 $a$ 包含系数，如 $(2x+y)^2$，学生常将其展开为 $2x^2+4xy+y^2$，忘记将系数 $2$ 也平方（应为 $4x^2$）。

## Parent Guidance
- 念熟顺口溜：让孩子背熟“首平方，尾平方，首尾两倍放中央”或者“符号同，取正；符号异，取负”。口诀能帮助在紧张考试中减少遗漏。
- 重视格式：刚开始练习时，强制要求孩子用括号把 $a$ 和 $b$ 括起来写展开式。比如 $(2x-3y)^2$，第一步必须写出：$(2x)^2 - 2(2x)(3y) + (3y)^2$，第二步再化简。
- 专项纠错：如果孩子写出 $(a+b)^2 = a^2+b^2$，不要马上批评，让他们代入数字验证，比如 $a=2, b=3$，算一算两边相等吗（$25 \neq 13$），建立自我查错意识。

## Linked Mistake Concepts
- 积的乘方与和的乘方混淆：$(ab)^2 = a^2b^2$ 是正确的，但 $(a+b)^2 \neq a^2+b^2$。乘法可以分配指数，加法绝对不行。
- 负号提取错误：处理 $(-x+y)(-x-y)$ 时，不明白哪一项是平方差公式中的“相同项”。

## Examples
```text
Question:
利用乘法公式计算下列各式：
(1) (3x - 2y)(3x + 2y)
(2) (-x - 4y)^2
(3) 102 \times 98

Solution:
(1) (3x - 2y)(3x + 2y)
  // 符合平方差公式 (a-b)(a+b) = a^2 - b^2，其中 a=3x, b=2y
  = (3x)^2 - (2y)^2
  = 9x^2 - 4y^2

(2) (-x - 4y)^2
  // 提取负号，(-x - 4y)^2 = [-(x + 4y)]^2 = (x + 4y)^2
  // 使用完全平方公式 (a+b)^2 = a^2 + 2ab + b^2
  = x^2 + 2(x)(4y) + (4y)^2
  = x^2 + 8xy + 16y^2

(3) 102 \times 98
  // 构造平方差形式
  = (100 + 2)(100 - 2)
  = 100^2 - 2^2
  = 10000 - 4
  = 9996
```
