# 方差与离差平方和(1)

## Core Idea
前面学习的平均数和中位数反映的是数据的“集中趋势”，而**方差（Variance）**反映的是数据的**“离散程度”（波动大小）**。
- **离差**：每一个数据 $x_i$ 与平均数 $\bar{x}$ 之间的差，即 $(x_i - \bar{x})$。
- **离差平方和**：把所有数据的离差平方后加起来，即 $(x_1-\bar{x})^2 + (x_2-\bar{x})^2 + \cdots + (x_n-\bar{x})^2$。
- **方差公式**：离差平方和的平均值。用 $s^2$ 表示。
  $$s^2 = \frac{1}{n} \left[ (x_1-\bar{x})^2 + (x_2-\bar{x})^2 + \cdots + (x_n-\bar{x})^2 \right]$$

**核心结论**：方差越大，说明数据偏离平均数越严重，数据波动越大，越不稳定；方差越小，数据波动越小，越稳定。

## Learning Steps
1. **理解波动的物理意义**：两个人平均分都是 80 分，但一个是(80,80,80)，一个是(60,80,100)，显然两者的稳定性不同。方差就是用来衡量这种不同的。
2. **掌握计算三部曲**：
   - 第一步：求出这组数据的平均数 $\bar{x}$。
   - 第二步：求出每个数据与平均数的差，并将其平方（求离差平方）。
   - 第三步：将所有的平方相加后，除以数据的总个数 $n$。
3. **应用决策**：在射击比赛、机床加工等需要“稳定性”的场景中，利用方差做出最佳选择（选方差小的）。

## Common Pitfalls
- **计算方差前算错平均数**：平均数是方差计算的基础，如果平均数算错了，后面的计算全盘皆输。
- **忘记除以 $n$**：算完了离差平方和，以为这就是结果，忘记了还要除以数据的个数求平均。
- **把“方差大”当成好事**：有些学生看到算出来的数字大，就直觉认为是好的。在统计学中，方差大代表不稳定、波动大，对于评价成绩或机器性能通常是“坏事”。

## Parent Guidance
- 让孩子把方差想象成“调皮度”。全班同学离老师（平均数）越远，这个班的“调皮度（方差）”就越大，纪律越差。所以方差越小越稳定，越听话。
- 检查孩子的草稿纸。算方差必须有条理，建议孩子列一个三行小表格：第一行写原数据，第二行写差值，第三行写平方，最后相加除以 $n$。这样能极大减少计算失误。

## Linked Mistake Concepts
- 平均数计算不准
- 负数的平方符号处理错误（例如把 $(-2)^2$ 算成 $-4$）
- 审题时没有抓住“求最稳定的”这一题眼

## Examples

**Example 1**
甲、乙两名射击运动员在相同的条件下各射击 5 次，成绩如下（单位：环）：
甲：8, 9, 7, 8, 8
乙：10, 7, 9, 6, 8
请计算两人的方差，并判断谁的成绩更稳定。

**Solution**
```text
第一步：求两人的平均数
甲的平均数 = (8 + 9 + 7 + 8 + 8) / 5 = 40 / 5 = 8。
乙的平均数 = (10 + 7 + 9 + 6 + 8) / 5 = 40 / 5 = 8。

第二步：求方差
甲的方差 s甲² = [(8-8)² + (9-8)² + (7-8)² + (8-8)² + (8-8)²] / 5
              = [0 + 1 + 1 + 0 + 0] / 5 = 2 / 5 = 0.4。
乙的方差 s乙² = [(10-8)² + (7-8)² + (9-8)² + (6-8)² + (8-8)²] / 5
              = [4 + 1 + 1 + 4 + 0] / 5 = 10 / 5 = 2。

第三步：比较
因为 s甲² = 0.4 < 2 = s乙²，
所以甲的方差更小。
答：甲的成绩更稳定。
```

**Example 2**
如果一组数据的方差为 $0$，这意味着什么？

**Solution**
```text
方差为 0 意味着离差平方和为 0，
即每一个离差 (xi - x̄)² 都必须等于 0。
这意味着这组数据中的每一个数都等于平均数。
结论：这组数据中的所有数值都完全相等，没有任何波动。
```
