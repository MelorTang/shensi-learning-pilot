# 正弦与余弦

## Core Idea
在直角三角形中，锐角 $A$ 的正弦（$\sin A$）定义为其对边与斜边的比；余弦（$\cos A$）定义为其邻边与斜边的比。它们反映了锐角与其对应边长的密切关系，且由于直角边总是小于斜边，所以对于任意锐角 $A$，$0 < \sin A < 1$ 且 $0 < \cos A < 1$。

## Learning Steps
1. 明确正弦和余弦的数学定义式：$\sin A = \frac{a}{c}$，$\cos A = \frac{b}{c}$。
2. 学习同一锐角的正弦平方和余弦平方的关系：$\sin^2 A + \cos^2 A = 1$。
3. 发现互余两角的三角比关系：若 $\angle A + \angle B = 90^\circ$，则 $\sin A = \cos B$，$\cos A = \sin B$。
4. 运用定义计算具体图形中的正弦和余弦值。

## Common Pitfalls
- 符号书写错误：写成 $sinA$（没有空格）或将平方写成 $\sin A^2$ 而不是 $\sin^2 A$。
- 求比值时不化简，或者忘记单位。虽然三角比没有单位，但在计算过程中如果边长带单位，必须被约掉。

## Parent Guidance
和孩子一起记忆口诀：“正弦是对斜，余弦是邻斜”。可以通过画不同的直角三角形，让孩子快速指认对边、邻边和斜边，加强条件反射。

## Linked Mistake Concepts
- 勾股定理与三角比的割裂：学生在已知正弦值求余弦值时，不善于利用勾股定理构建一个辅助直角三角形来求解，而是试图死记硬背公式。
- 取值范围的忽略：在解题中求出 $\sin A = 1.5$ 却没有发现错误，未牢记正弦和余弦值必小于 1。

## Examples
```text
题目：
已知在直角 △ABC 中，∠C = 90°，sin A = 5/13，求 cos A 和 sin B。

解答：
因为 sin A = 对边 / 斜边 = 5/13，
我们可以设 ∠A 的对边 a = 5k，斜边 c = 13k (k > 0)。
根据勾股定理：
b^2 = c^2 - a^2
b^2 = (13k)^2 - (5k)^2 = 169k^2 - 25k^2 = 144k^2
所以 ∠A 的邻边 b = 12k。

1) 求 cos A：
cos A = 邻边 / 斜边 = 12k / 13k = 12/13。

2) 求 sin B：
∠B 的对边是 b，斜边是 c。
sin B = 对边 / 斜边 = b / c = 12k / 13k = 12/13。

（注：也可直接利用互余两角关系，因为 ∠A + ∠B = 90°，所以 sin B = cos A = 12/13）
```
