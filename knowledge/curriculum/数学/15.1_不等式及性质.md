# 不等式及性质

## Core Idea
不等式是表示不相等关系的代数式，主要包括“>”、“<”、“≥”、“≤”和“≠”。它的核心在于刻画数量之间的大小关系。不等式的性质是不等式变形的基础：不等式两边同加或同减去同一个数或式子，不等号方向不变；两边同乘或同除以同一个正数，不等号方向不变；两边同乘或同除以同一个负数，不等号方向改变。这是与等式性质最根本的区别。

## Learning Steps
1. **认识不等式的意义**：理解大于、小于、不大于（小于或等于）、不小于（大于或等于）的符号表示。能将生活中的不等关系转化为数学符号。
2. **掌握不等式的基本性质**：
   - 性质1：如果 a > b，那么 a ± c > b ± c。
   - 性质2：如果 a > b，且 c > 0，那么 ac > bc，a/c > b/c。
   - 性质3：如果 a > b，且 c < 0，那么 ac < bc，a/c < b/c。
3. **应用性质进行变形**：通过不等式的性质，把复杂的不等式化简为 x > a 或 x < a 的形式。

## Common Pitfalls
- **性质3的遗忘**：在不等式两边同乘或同除以一个负数时，忘记改变不等号的方向。
- **隐含条件的忽略**：在涉及除以含有字母的式子时，不考虑字母的符号，直接同除。
- **不大于与不小于的混淆**：“不大于”对应“≤”，“不小于”对应“≥”，容易只写“<”或“>”。

## Parent Guidance
家长可以通过日常生活中的实例（如购物预算、身高要求等）帮助孩子理解“至少”、“最多”这类词语在不等式中的数学表达。重点监督孩子在进行性质3的变形时，是否每次都做到了“变号”。可以要求孩子在遇到乘除负数时，先用红笔画出负号，作为警醒。

## Linked Mistake Concepts
- 等式的性质（容易将不等式性质3与等式性质混淆，等式中无需变号）
- 负数大小比较（负数绝对值大的反而小，这是导致性质3变号的根本原因）
- 字母表示数（当c为未知数时，不能随意进行不等式的除法）

## Examples
题目：将不等式 -2x + 3 > 7 变形为 x < a 的形式。
```text
解：
根据不等式的性质1，不等式两边同减去3，得：
-2x > 7 - 3
-2x > 4
根据不等式的性质3，不等式两边同除以-2，改变不等号的方向，得：
x < 4 / (-2)
x < -2
```
