# 圆与扇形的面积

## Core Idea
掌握圆面积公式 $S=\pi r^2$ 及其推导，理解扇形面积公式 $S=\frac{n\pi r^2}{360}$ 及其变形 $S=\frac{1}{2}lr$。能够利用分割、拼接的方法计算不规则图形的面积。

## Learning Steps
1. 圆面积公式的推导：利用割补法，将圆近似转化为矩形来理解面积公式。
2. 掌握扇形面积的两种计算方法：通过圆心角计算和通过弧长计算。
3. 学习计算组合图形的面积：将复杂图形分解为圆、扇形、多边形等简单图形的加减。

## Common Pitfalls
- 计算扇形面积时弄错圆心角，或者误用圆的周长公式代替面积。
- 公式中的平方计算错误，例如把 $r^2$ 算成 $2r$。

## Parent Guidance
用切比萨饼的例子解释扇形的面积，帮助孩子建立直观的面积分割模型。可以让孩子观察家里不同大小的圆形盘子，讨论面积和半径的平方成正比。

## Linked Mistake Concepts
- 扇形面积与弧长公式混淆，特别是容易把底部的360记成180。
- 在计算半圆面积时，忘记最后要除以2。

## Examples
题目：
已知一个扇形的圆心角为 $120^\circ$，半径为 $3\text{ cm}$。求该扇形的面积。

解答：
```text
扇形面积 S = (n * π * r^2) / 360 
          = (120 * π * 3^2) / 360 
          = (120 * π * 9) / 360 
          = 3π cm²。
```
