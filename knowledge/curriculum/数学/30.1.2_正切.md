# 正切

## Core Idea
在直角三角形中，锐角 $A$ 的正切（$\tan A$）定义为其对边与邻边的比。与正弦和余弦不同，正切的值可以大于 1，也可以小于 1，且随着锐角 $A$ 的增大，$\tan A$ 的值也逐渐增大。正切常用于表示坡度或倾斜程度。

## Learning Steps
1. 明确正切的定义式：$\tan A = \frac{a}{b}$（对边/邻边）。
2. 探索正切与正弦、余弦之间的关系：$\tan A = \frac{\sin A}{\cos A}$。
3. 学习坡度和坡角的概念：坡度 $i = \tan \alpha = \frac{\text{垂直高度}}{\text{水平宽度}}$。
4. 运用正切知识解决实际的测高、测距问题。

## Common Pitfalls
- 求正切时将对边和邻边放反，写成了 $\frac{\text{邻边}}{\text{对边}}$。
- 在实际应用（如坡度问题）中，混淆了坡度（比值）和坡角（度数），错误地将坡度写成度数。

## Parent Guidance
利用坡度这个生活概念，比如盘山公路或滑梯的陡缓，向孩子解释正切的意义。可以告诉孩子，正切值越大，路就越陡。

## Linked Mistake Concepts
- 认为正切值也不能大于 1。由于直角三角形中对边可以大于邻边，所以正切值完全可以大于 1。
- 角度的增大与三角比的增减性混淆：对于锐角，角度越大，正弦和正切值越大，但余弦值却越小。

## Examples
```text
题目：
一段斜坡的垂直高度为 3 米，水平宽度为 4 米。
(1) 求这段斜坡的坡度 i；
(2) 若设坡角为 α，求 tan α 和 sin α。

解答：
(1) 坡度 i 是垂直高度与水平宽度的比值。
i = 垂直高度 / 水平宽度 = 3 / 4 = 1 : 1.33 （或写成 3:4, 0.75）。

(2) 坡角 α 所在的直角三角形中，对边 a = 3，邻边 b = 4。
tan α = 对边 / 邻边 = 3 / 4。

利用勾股定理求出斜边 c：
c = √(3^2 + 4^2) = √(9 + 16) = √25 = 5。

sin α = 对边 / 斜边 = 3 / 5。
```
