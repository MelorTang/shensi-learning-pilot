# 等腰三角形的判定

## Core Idea
等腰三角形的判定主要解决“如何证明一个三角形是等腰三角形”的问题。核心方法是“等角对等边”（如果一个三角形有两个角相等，那么这两个角所对的边也相等）。这是将“角的相等关系”转化为“边的相等关系”的桥梁。

## Learning Steps
1. **掌握等角对等边定理**：理解在同一个三角形中，有两个角相等，就能推导出它是一个等腰三角形。
2. **结合平行线构造型**：熟练处理“平行线 + 角平分线 = 等腰三角形”这一经典几何模型。
3. **综合运用性质与判定**：在复杂的几何证明中，先通过角的关系证明是等腰三角形，然后再利用等腰三角形的“三线合一”解决更深层的问题。

## Common Pitfalls
- **在不同三角形中乱用**：“等角对等边”必须是在“同一个三角形”中才成立，学生常把两个全等三角形中的对应角相等当成等腰三角形的判定。
- **因果倒置**：需要证明是等腰三角形时，却不知不觉用上了等腰三角形的性质（如三线合一）作为已知条件。

## Parent Guidance
帮助孩子总结经典模型，比如当题目里同时出现了“角平分线”和“平行线”时，这几乎是在明示“这里藏着一个等腰三角形”。家长可以引导孩子通过“内错角相等”或“同位角相等”做传递，把角的关系转移到同一个三角形内。

## Linked Mistake Concepts
- 平行线的性质（判定定理常与平行线结合命题）
- 角平分线的定义（也是构造相等角的重要来源）

## Examples
题目：如图，在△ABC中，AB=AC，BD平分∠ABC交AC于点D，若过点D作DE//BC交AB于点E，求证：△BDE是等腰三角形。
```text
证明：
因为 DE // BC (已知)，
所以 ∠EDB = ∠DBC (两直线平行，内错角相等)。
因为 BD平分∠ABC (已知)，
所以 ∠EBD = ∠DBC (角平分线的定义)。
所以 ∠EDB = ∠EBD (等量代换)。
在 △BDE 中，有两个内角相等 (∠EDB = ∠EBD)，
所以 EB = ED (等角对等边)。
所以 △BDE 是等腰三角形。
```
