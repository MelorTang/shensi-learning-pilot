# 圆内接四边形

## Core Idea
如果一个四边形的四个顶点都在同一个圆上，这个四边形叫做**圆内接四边形**，这个圆叫做四边形的外接圆。
**圆内接四边形的重要性质**：
1. **对角互补**：圆内接四边形的对角之和等于 $180^\circ$。（即 $\angle A + \angle C = 180^\circ$，$\angle B + \angle D = 180^\circ$）
2. **外角等于内对角**：圆内接四边形的一个外角等于它的内对角。（即外角等于与它相邻内角的对角）

利用这两个性质，可以在证明题或计算题中快速进行角度的转换。

## Learning Steps
1. **辨认圆内接四边形**：四个顶点必须全在圆上，缺一不可。
2. **理解定理证明**：连接对角线，利用同弧所对的圆周角相等，或者连接圆心利用圆心角之和为 $360^\circ$ 且等于圆周角的两倍，来证明对角互补。
3. **应用性质转换角度**：在解题中看到圆内接四边形，条件反射第一步写出对角互补。
4. **掌握外角转化**：理解“外角等于内对角”是“对角互补”加上“平角为 $180^\circ$”的直接推论，能直接使用可以大大节省步骤。

## Common Pitfalls
- **对角互补用错对象**：有时只有三个点在圆上，一个点在圆内，学生也想当然地使用“对角互补”。
- **混淆“圆内接”与“圆外切”四边形的性质**：
  - 圆内接四边形是**对角互补**。
  - 圆外切四边形是**对边之和相等**（利用切线长定理）。两者极易搞混！
- 计算时加减出错，如把 $180^\circ - 70^\circ$ 算错。

## Parent Guidance
- 可以问孩子这样一个问题：“平行四边形能不能内接于圆？”引导孩子思考：平行四边形对角相等，如果是圆内接四边形对角又必须互补，那么对角只能是 $90^\circ$。所以只有矩形和正方形能内接于圆！
- 强调“内接四边形看角度，外切四边形看边长”这个记忆口诀，防止性质混淆。

## Linked Mistake Concepts
- 平行四边形的性质冲突理解不深
- “圆内接”与“圆外切”概念与图形的混淆
- 平角概念和外角定义的薄弱

## Examples

**Example 1**
四边形 $ABCD$ 内接于 $\odot O$，若 $\angle A = 50^\circ$，求 $\angle C$ 的度数。

**Solution**
```text
因为四边形 ABCD 是 ⊙O 的内接四边形，
根据圆内接四边形的对角互补性质：
∠A + ∠C = 180°。
因为 ∠A = 50°，
所以 ∠C = 180° - 50° = 130°。
```

**Example 2**
已知四边形 $ABCD$ 内接于圆，它的一个外角 $\angle CDE = 75^\circ$，求 $\angle B$ 的度数。

**Solution**
```text
因为四边形 ABCD 内接于圆，
根据圆内接四边形的性质：任意一个外角等于它的内对角。
∠CDE 是顶点 D 处的外角，它的相邻内角是 ∠ADC，∠ADC 的对角是 ∠B。
所以 ∠B = ∠CDE = 75°。

(另一种推导：∠ADC = 180° - 75° = 105°，因为对角互补，∠B = 180° - 105° = 75°。)
```
