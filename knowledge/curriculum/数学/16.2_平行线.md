# 平行线

## Core Idea
平行线是同一平面内不相交的两条直线。平行线的性质和判定是平面几何的核心内容之一。判定定理是“由角定线”（通过角的关系证明直线平行），而性质定理是“由线定角”（通过直线平行推导角的关系）。这两者的逻辑方向正好相反，是几何推理中相互依存的工具。

## Learning Steps
1. **理解平行线的概念和平行公理**：同一平面内两条不相交的直线。经过直线外一点，有且只有一条直线与这条直线平行。
2. **掌握三线八角**：能在复杂图形中准确识别同位角、内错角、同旁内角（通常以“F”、“Z”、“U”字型辅助记忆）。
3. **学习平行线的判定**：同位角相等，两直线平行；内错角相等，两直线平行；同旁内角互补，两直线平行。
4. **学习平行线的性质**：两直线平行，同位角相等；两直线平行，内错角相等；两直线平行，同旁内角互补。

## Common Pitfalls
- **性质与判定混淆**：在证明过程中，把“因为平行，所以角相等”错写成判定定理的理由。
- **找错“三线八角”**：特别是截线找错，导致把毫无关系的角当成同位角或内错角。
- **前提条件缺失**：所有的判定和性质都必须指明“哪两条直线被哪条直线所截”。

## Parent Guidance
平行线是初中几何的第一道大坎。家长可以用彩笔帮助孩子在题目上描出构成角的边，形成“F”、“Z”或“U”型，从而准确找到关系角。在书写证明过程时，一定要让孩子标明前提，例如“因为 AB // CD (已知)”。

## Linked Mistake Concepts
- 对顶角和邻补角（平行线中找角时，往往需要借助对顶角和邻补角进行等量代换）
- 相交线（平行是相交的对立面，但性质判定都是依靠相交线形成的角）

## Examples
题目：如图，已知∠1 = 60°，∠2 = 60°，∠3 = 100°，求∠4的度数。
(图形描述：直线a, b被截线c所截，∠1和∠2是同位角，∠3和∠4是同旁内角。)
```text
解：
因为 ∠1 = 60°，∠2 = 60° (已知)，
所以 ∠1 = ∠2。
所以 直线a // 直线b (同位角相等，两直线平行)。
因为 a // b (已证)，
所以 ∠3 + ∠4 = 180° (两直线平行，同旁内角互补)。
因为 ∠3 = 100° (已知)，
所以 ∠4 = 180° - 100° = 80°。
答：∠4的度数为80°。
```
