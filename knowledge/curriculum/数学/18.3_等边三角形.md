# 等边三角形

## Core Idea
等边三角形是三条边都相等的特殊等腰三角形，它是最具对称美的多边形之一。它的核心特征在于高度的统一：三个内角都是60°，任意一边上的中线、高、角平分线都“三线合一”。此外，直角三角形中30°角所对的直角边等于斜边的一半，也是本节的重点定理（由等边三角形推导而来）。

## Learning Steps
1. **掌握等边三角形的性质**：三边相等，三个内角都是60°，具有三条对称轴。
2. **掌握等边三角形的判定**：三边相等的三角形是等边三角形；三个角相等的三角形是等边三角形；有一个角是60°的等腰三角形是等边三角形。
3. **理解并应用30°角直角三角形性质**：在直角三角形中，如果有一个锐角是30°，那么它所对的直角边等于斜边的一半。

## Common Pitfalls
- **判定条件缺失**：证明“有一个角是60°的等腰三角形是等边三角形”时，忘记先证明它是等腰三角形，只看到有个60°角就下结论。
- **30°定理用错对象**：在非直角三角形中看到30°角，就直接得出“对边等于另一边的一半”的错误结论。30°定理必须在“直角三角形”这个大前提下才成立。

## Parent Guidance
家长可以鼓励孩子用直尺和圆规自己画一个完美的等边三角形（作已知线段的垂直平分线相关画法），体会其对称之美。在做含有30°角的题目时，一定要让孩子先圈出题目中的“直角”或者“⊥”符号，确认是直角三角形后再使用定理，避免生搬硬套。

## Linked Mistake Concepts
- 直角三角形的概念（30°角性质的必须前提）
- 等腰三角形的性质与判定（等边三角形是等腰三角形的升级版）

## Examples
题目：在△ABC中，AB=AC，∠A=60°。证明：△ABC是等边三角形。
```text
证明：
因为 AB = AC (已知)，
所以 △ABC 是等腰三角形，且 ∠B = ∠C (等边对等角)。
根据三角形内角和定理：
∠A + ∠B + ∠C = 180°
60° + 2∠B = 180°
2∠B = 120°
∠B = 60°
所以 ∠C = 60°。
因为 ∠A = ∠B = ∠C = 60°，
所以 △ABC 是等边三角形 (三个角都相等的三角形是等边三角形)。
```
