# 中心投影

## Core Idea
中心投影是由一个点光源发出的光线照射物体所形成的投影。它的主要特点是：所有的投射线都交于一点（即光源）。在中心投影下，物体离光源越近，其投影通常越大；离光源越远，投影越小。这种投影方式符合人类眼睛观察世界的透视原理。

## Learning Steps
1. 认识中心投影的基本构成：点光源、物体、投影面。
2. 学习如何在已知光源位置的情况下，画出简单物体（如线段、三角形）的中心投影。
3. 学习逆向问题：如何通过物体及其中心投影的位置，反向寻找点光源的位置（通常是连接物体顶端与投影顶端，其延长线的交点即为光源）。
4. 探究在中心投影下，线段长度与投影长度的比例关系（通常利用相似三角形来解决）。

## Common Pitfalls
- 寻找光源位置时，只连接了一组对应点，没有连接两组以上对应点来找交点，导致光源位置找错或无法确定。
- 在计算中心投影的长度时，误用平行投影的比例关系，没有画出剖面图利用相似三角形求解。

## Parent Guidance
可以拿一个手电筒和一块橡皮擦，在墙面上给孩子演示。改变手电筒与橡皮擦的距离，让孩子观察影子的放大和缩小；改变手电筒的角度，观察影子形状的变化。

## Linked Mistake Concepts
- 对透视变形的不适应：中心投影中，互相平行的线段如果不平行于投影面，它们的投影可能不再平行，而是相交于“灭点”。初中生容易认为平行的线段投影后一定还是平行的。
- 认为中心投影一定比原物体大。这取决于物体、光源和投影面的相对位置，虽然通常生活经验是放大，但在特定几何布置下也可能不是。

## Examples
```text
题目：
晚上，小明在路灯下散步。已知小明身高为 1.6 米，当他距离路灯底部 3 米时，他在地上的影长为 2 米。求路灯的高度。

解答：
设路灯的顶点为点 S，路灯底部为点 O。路灯高度为 h (SO = h)。
小明的头顶为点 A，脚底为点 B。小明身高 AB = 1.6 米。
小明的影子顶端为点 C。影长 BC = 2 米。
根据题意，小明距离路灯底部 OB = 3 米。

由中心投影的特性可知，光线 SA 经过头顶 A 到达影子顶端 C，即 S, A, C 三点共线。
因为小明 (AB) 和路灯 (SO) 都垂直于地面，所以 AB // SO。
因此，△CAB ∽ △CSO。

根据相似三角形对应边成比例：
CB / CO = AB / SO
注意，CO = CB + BO = 2 + 3 = 5 米。

代入数据：
2 / 5 = 1.6 / h
解得：
2h = 5 * 1.6
2h = 8
h = 4

答：路灯的高度为 4 米。
```
