# 认识二元一次方程组

## Core Idea
理解二元一次方程、二元一次方程组及其解的概念，体会多元问题可以通过设多个未知数、列多个方程来表达，建立代数模型思想。

## Learning Steps
1. 理解二元一次方程的定义：含有两个未知数，且未知数项的最高次数为1的整式方程。
2. 认识二元一次方程组：把含有相同未知数的两个二元一次方程联立在一起。
3. 检验给定数值是否为二元一次方程或二元一次方程组的解。

## Common Pitfalls
- 误以为含有两个未知数的方程组就一定是二元一次方程组（例如出现了分母中含有未知数的情况，或者未知数相乘的情况 $xy$）。
- 代入数值检验时，只代入了一个方程而没有代入另一个方程进行验证。

## Parent Guidance
举一些生活中买两种东西的例子，比如“买2支铅笔和3块橡皮一共花了5元，铅笔和橡皮的单价分别是多少”，引导孩子体会为什么需要列出两个关系式，感受方程组的实际应用价值。

## Linked Mistake Concepts
- 将二元一次方程的解误认为只有唯一的一组，而不明白单个二元一次方程通常有无数组解。
- 认为未知数的系数一定要是整数，遇到分数系数时不知所措。

## Examples
题目：
判断下面哪组数值是二元一次方程组
{ 2x + y = 5
{ x - y = 1
的解？
A. x = 1, y = 3
B. x = 2, y = 1

解答：
```text
将 A 组 x=1, y=3 代入：
第一个方程: 2(1) + 3 = 5，成立。
第二个方程: 1 - 3 = -2 ≠ 1，不成立。

将 B 组 x=2, y=1 代入：
第一个方程: 2(2) + 1 = 5，成立。
第二个方程: 2 - 1 = 1，成立。

所以 B 组 (x=2, y=1) 是该方程组的解。
```
