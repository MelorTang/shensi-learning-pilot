# 解直角三角形

## Core Idea
本节是对解直角三角形基本技能的强化训练。核心在于熟练穿梭于三条边和两个锐角之间，根据题目的已知条件，灵活选择最直接、计算量最小的关系式。

## Learning Steps
1. 熟练掌握六个基本元素（三条边 $a,b,c$ 和三个角 $\angle A, \angle B, \angle C$）中独立元素的数量关系。
2. 针对不同的已知条件组合进行专项训练：
   - 类型一：已知两条直角边
   - 类型二：已知一直角边和斜边
   - 类型三：已知一锐角和一直角边
   - 类型四：已知一锐角和斜边
3. 学习检查结果的合理性：例如大角对大边、直角边必须小于斜边。

## Common Pitfalls
- 在求边长时，选错了三角函数。例如已知对边求斜边，应该用正弦，却用成了正切，导致求出的是邻边。
- 解题步骤不规范，没有写明“在 Rt△ABC 中”作为大前提。

## Parent Guidance
如果孩子在这部分做题速度慢，家长可以和孩子一起列一张关系网图，把所有的公式（勾股定理、面积法、三角函数）连起来，形成解题的“工具箱”。

## Linked Mistake Concepts
- 迷信某一种方法：例如凡是求边都只用勾股定理，不去尝试用三角函数，导致在只知一边一角时束手无策。
- 将 $\angle A$ 的正弦公式错套在 $\angle B$ 上，未根据具体的角重新确定对边和邻边。

## Examples
```text
题目：
在 Rt△ABC 中，∠C = 90°。已知 a = 5，c = 13，求 b 及 sin A, tan A 的值。

解答：
在 Rt△ABC 中，已知 a（对边）= 5，c（斜边）= 13。

1) 求直角边 b：
根据勾股定理 a^2 + b^2 = c^2：
5^2 + b^2 = 13^2
25 + b^2 = 169
b^2 = 144
b = 12

2) 求 sin A：
sin A = a / c = 5 / 13

3) 求 tan A：
tan A = a / b = 5 / 12

综上，b = 12，sin A = 5/13，tan A = 5/12。
```
