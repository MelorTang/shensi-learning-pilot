# 平移

## Core Idea
平移（Translation）的概念：在平面内，将一个图形沿某个方向移动一定的距离，这样的图形运动称为平移。
平移的三大基本性质：
1. **形状和大小不变**：平移前后的两个图形全等。
2. **对应线段平行（或共线）且相等**。
3. **对应点的连线平行（或共线）且相等**，且长度等于平移的距离，方向即为平移的方向。
平移不仅可以看作是图形的运动，也是研究复杂几何图形（特别是平行线和线段等量关系）的辅助线构造利器。

## Learning Steps
1. 识别平移现象：从生活中提取平移现象（如电梯升降、推拉窗、抽屉），并将其与旋转、翻折区分开。
2. 掌握平移的两个要素：**平移方向**和**平移距离**。描述一个平移必须同时具备这两个要素。
3. 利用网格画平移：在方格纸上，通过将图形的各个顶点按指定方向和距离分别平移，然后依次连接，画出平移后的图形。
4. 应用性质解题：利用平移前后“对应线段平行且相等”的性质，将分散在图形不同位置的线段集中到同一个三角形或多边形中，从而求角度、长度或面积。

## Common Pitfalls
- 混淆平移距离：部分学生认为图形之间的最近距离（如两个多边形边界的距离）是平移距离。实际上，**平移距离必须是一对对应点之间的线段长度**。
- 平移方向描述不准：说“向右平移”或“沿某条线平移”而忘记说明具体长度，或者指定了长度却未指明沿着哪条线的哪个方向。
- 对应关系找错：在复杂的重叠图形中进行平移后，找不到哪些点是原图形的对应点，导致对应线段找错。

## Parent Guidance
- 动手操作感受平移：找一张半透明的纸（或烘焙纸），在上面画一个三角形，盖在一张白纸上。告诉孩子，只要纸不转动，无论怎么平放着滑来滑去，这个过程就是平移。一旦纸转了，就不是平移了。
- 面积割补法的妙用：给孩子展示一道经典题：求长方形草坪里有一条弯曲小路（宽度一致）后的剩余草坪面积。提示孩子将弯曲小路两边的草坪沿着小路方向直接“平移”拼在一起，小路的弯曲度完全不影响计算，深刻体会平移不变形、不改变面积的威力。

## Linked Mistake Concepts
- 与平移坐标公式混淆：到八年级学了平面直角坐标系后，会有“上加下减，左减右加”的坐标变换口诀。在此阶段，学生如果不借用坐标系，仅凭纯几何思维，可能会觉得无从下手。

## Examples
```text
Question:
如图（想象），将线段 AB 沿水平方向向右平移 3cm 得到线段 A'B'。
已知 A、B 两点之间的距离为 5cm。
(1) 线段 A'B' 的长度是多少？
(2) 连接 AA' 和 BB'，线段 AA' 和 BB' 有什么关系？它们的长度分别是多少？
(3) 四边形 ABB'A' 是什么图形？

Solution:
(1) 根据平移的基本性质 1：“平移前后的图形全等，线段长度不变”。
所以 A'B' = AB = 5cm。

(2) 根据平移的基本性质 3：“对应点的连线平行（或共线）且相等，长度等于平移的距离”。
AA' 是点 A 平移到 A' 的轨迹，BB' 是点 B 平移到 B' 的轨迹。
所以 AA' // BB'，且 AA' = BB'。
因为平移的距离是 3cm，所以 AA' = 3cm，BB' = 3cm。

(3) 因为 AA' // BB' 且 AA' = BB'，
有一组对边平行且相等的四边形是平行四边形。
所以四边形 ABB'A' 是平行四边形。
```
