# 命题与证明

## Core Idea
命题是判断一件事情的语句，分为真命题和假命题。证明是从已知条件出发，运用定义、公理、已证定理，进行有理有据的逻辑推理，最终得出结论的过程。这一节是从“直观感受”向“逻辑演绎”转变的关键里程碑，标志着数学思维的严密化。

## Learning Steps
1. **理解命题的概念**：能判断一个句子是不是命题，并找出命题的“题设（条件）”和“结论”。
2. **改写命题形式**：能把命题改写成“如果...那么...”的形式，明确因果关系。
3. **区分真假命题**：判断真假命题，并且知道要说明一个命题是假命题，只需举一个反例即可。
4. **掌握证明的步骤**：根据题意画出图形，结合图形写出“已知”和“求证”，最后写出逻辑严密的“证明”过程。

## Common Pitfalls
- **题设和结论找错**：特别是在省略了“如果”或“那么”的简短命题中，容易混淆条件和结论。
- **循环论证**：在证明过程中，把要证明的结论当作了已知条件来用。
- **举反例不准确**：为了证明假命题，举的反例虽然结论不成立，但也并不满足题设。

## Parent Guidance
这部分内容看似偏文科（咬文嚼字），实则是逻辑训练的核心。家长可以和孩子玩“造句”游戏，把生活中的规律改成“如果...那么...”的形式，并讨论真假。对于书面证明，一定要强迫孩子写出“已知”和“求证”，规范书面格式，这是几何学习的“规矩”。

## Linked Mistake Concepts
- 平行线的性质与判定（常作为证明题的基础素材）
- 逻辑学基础（如逆命题的概念，虽然初中较少深入，但思维方式一致）

## Examples
题目：将命题“对顶角相等”改写成“如果...那么...”的形式，并指出它的题设和结论。
```text
解：
改写：如果两个角是对顶角，那么这两个角相等。
题设：两个角是对顶角。
结论：这两个角相等。
此命题是一个真命题。
```
