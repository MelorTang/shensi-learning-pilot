# 解直角三角形

## Core Idea
“解直角三角形”是指在已知直角三角形的几个元素（边和锐角）的情况下，求出其余未知元素的过程。根据“知二求三”（其中至少有一条边）的原则，通过综合运用勾股定理、三角函数定义以及直角三角形两锐角互余的性质，可以解出整个三角形。

## Learning Steps
1. 梳理直角三角形中的三类关系：
   - 边与边：勾股定理 $a^2 + b^2 = c^2$
   - 角与角：$\angle A + \angle B = 90^\circ$
   - 边与角：三角函数（$\sin, \cos, \tan$）
2. 学习“已知两边解直角三角形”的方法（先用勾股定理解第三边或先用三角函数求角度）。
3. 学习“已知一边一锐角解直角三角形”的方法。
4. 总结解直角三角形的一般步骤：画草图 -> 标已知 -> 选公式 -> 求解。

## Common Pitfalls
- 在没有给出图形时，未明确哪一条边是斜边（如未说明 $\angle C = 90^\circ$），盲目套用公式导致错解。
- 滥用计算器，在要求精确解的题目中使用了近似小数（如用 $1.414$ 代替 $\sqrt{2}$）。

## Parent Guidance
提醒孩子在解题时，尽量使用题目直接给出的原始数据进行计算，少用中间求出的数据（如果求错了会导致步步错），这样可以减少累积误差。

## Linked Mistake Concepts
- 非直角三角形的硬套：题目给出的不是直角三角形，却直接使用正弦和余弦定理（初中尚未学习），或者直接当成直角三角形来解。
- 忽略多解情况：当已知两边，但未指明是直角边还是斜边时，应当分类讨论。

## Examples
```text
题目：
在直角 △ABC 中，∠C = 90°，已知 AC = 4，∠A = 60°。解这个直角三角形（即求出未知的边和角）。

解答：
已知：∠C = 90°，AC = 4，∠A = 60°。
未知元素有：∠B，边 AB，边 BC。

1) 求 ∠B：
因为直角三角形两锐角互余，
∠B = 90° - ∠A = 90° - 60° = 30°。

2) 求 BC（对边）：
可以使用正切函数：tan A = BC / AC
tan 60° = BC / 4
√3 = BC / 4
所以 BC = 4√3。

3) 求 AB（斜边）：
可以使用余弦函数：cos A = AC / AB
cos 60° = 4 / AB
1/2 = 4 / AB
AB = 8。
（也可以使用勾股定理：AB = √(AC^2 + BC^2) = √(16 + 48) = √64 = 8）

结论：∠B = 30°，BC = 4√3，AB = 8。
```
