# 求给定锐角的正弦、余弦与正切值

## Core Idea
特定锐角（如 $30^\circ, 45^\circ, 60^\circ$）的三角比是固定的精确值，这些被称为特殊角的三角函数值。熟练记忆并运用这些特殊值，是解决几何计算题和简化表达式的基础。对于非特殊角，通常需要使用科学计算器。

## Learning Steps
1. 利用含 $45^\circ$ 的等腰直角三角形，推导 $45^\circ$ 的正弦、余弦和正切值。
2. 利用含 $30^\circ, 60^\circ$ 的直角三角形（由等边三角形对折得来），推导这两个角的三角比值。
3. 制作并背诵特殊角三角函数值表。
4. 学习含有特殊角三角函数值的混合运算。
5. （可选）学习如何使用计算器求任意锐角的三角比，以及已知比值求角度（反三角函数）。

## Common Pitfalls
- 特殊角的值记忆混乱，例如把 $\sin 30^\circ$ 记成 $\frac{\sqrt{3}}{2}$，把 $\tan 60^\circ$ 记成 $\frac{\sqrt{3}}{3}$。
- 在进行代数运算时，带有根号的化简不彻底（分母未有理化）。

## Parent Guidance
可以教孩子一个记忆小窍门：$\sin$ 值的分母都是 2，分子分别是 $\sqrt{1}, \sqrt{2}, \sqrt{3}$（对应 $30^\circ, 45^\circ, 60^\circ$）；而 $\cos$ 值正好相反。$\tan$ 则是 $\sin$ 除以 $\cos$。抽查背诵非常有效。

## Linked Mistake Concepts
- 将三角函数符号当成普通变量：如认为 $\sin(A+B) = \sin A + \sin B$，这是绝对错误的。
- 不会灵活运用特殊角：当题目中出现 $\sqrt{3}$ 等数字时，不能敏锐地联想到 $30^\circ$ 或 $60^\circ$ 角。

## Examples
```text
题目：
计算：sin 30° * cos 60° + tan 45° - sin^2 45°

解答：
首先写出各特殊角的三角函数值：
sin 30° = 1/2
cos 60° = 1/2
tan 45° = 1
sin 45° = √2 / 2

代入原式：
原式 = (1/2) * (1/2) + 1 - (√2 / 2)^2
原式 = 1/4 + 1 - (2 / 4)
原式 = 1/4 + 1 - 1/2
原式 = 1/4 + 4/4 - 2/4
原式 = 3/4

所以，计算结果为 3/4。
```
