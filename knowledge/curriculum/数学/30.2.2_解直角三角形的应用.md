# 解直角三角形的应用

## Core Idea
将实际生活中的测高、测距、航海航行等问题，通过添加辅助线抽象为直角三角形的几何模型，并利用解直角三角形的知识求解。核心是“数学建模”和“辅助线的作法”。

## Learning Steps
1. 掌握实际应用中的专用名词术语：仰角、俯角、方位角、坡角、坡度等。
2. 学习如何根据题意画出准确的平面几何草图。
3. 学习构造直角三角形的常用技巧：作高线（垂线）。通常是将非直角三角形分割成两个直角三角形。
4. 在构造出的直角三角形中运用三角函数和勾股定理解题，最后还原为实际问题的答案。

## Common Pitfalls
- 仰角和俯角的定义不清，常把俯角（视线与水平线的夹角）画成视线与铅垂线的夹角。
- 方位角（如“北偏东 30°”）识别错误，画图时标错角的位置。
- 实际问题的最后答语中忘记带上单位。

## Parent Guidance
带孩子到户外，指着高楼或树木，用手势演示“仰角”和“俯角”的区别。可以让孩子尝试估算一个仰角，体会数学在实际测量中的力量。

## Linked Mistake Concepts
- 辅助线作法不当：在钝角三角形中作高时，往往忘记高在三角形的外部，导致图形画错。
- 在“双直角三角形”（有公共边）的问题中，不会利用公共边作为桥梁来联立方程求解，而是孤立地看两个三角形。

## Examples
```text
题目：
某人在 A 处测得某塔塔顶的仰角为 30°，他向塔的方向前进 20 米到达 B 处，此时测得塔顶的仰角为 60°。求该塔的高度（忽略人的身高，结果保留根号）。

解答：
设塔顶为点 C，塔底为点 D。CD 垂直于地面。
设塔高 CD = h 米。
在 Rt△ADC 中，∠DAC = 30°。
在 Rt△BDC 中，∠DBC = 60°。
已知 AB = 20 米。

根据正切定义：
在 Rt△BDC 中，tan 60° = CD / BD  =>  BD = h / tan 60° = h / √3
在 Rt△ADC 中，tan 30° = CD / AD  =>  AD = h / tan 30° = h / (√3/3) = h√3

根据线段关系：AD - BD = AB
所以：h√3 - h / √3 = 20
通分： (3h - h) / √3 = 20
2h / √3 = 20
2h = 20√3
h = 10√3

答：该塔的高度为 10√3 米。
```
