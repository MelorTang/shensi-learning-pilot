# 一次函数求值

## Core Idea

一次函数求值是把给定的 `x` 代入 `y = kx + b`，再按运算顺序求出对应的 `y`。关键是代入完整、符号清楚、运算顺序正确。

## Learning Steps

- 找到函数解析式中的 `k` 和 `b`。
- 确认题目给出的 `x`。
- 把 `x` 整体代入解析式，负数要加括号。
- 先算乘法，再算加减。
- 写出最终的 `y` 值。

## Common Pitfalls

- 负数代入时漏写括号。
- 把 `-3x` 看成 `-3 + x`。
- 先做加减，后做乘法。
- 抄错 `k` 或 `b` 的符号。

## Examples

### Example 1

题目：已知 `y = -3x + 2`，当 `x = -2` 时，求 `y`。

解法：

```text
y = -3 × (-2) + 2
y = 6 + 2
y = 8
```

## Parent Guidance

让孩子先口头说出“代入、乘法、加减”三步，再动笔。遇到负数代入时，要求孩子先写括号。
