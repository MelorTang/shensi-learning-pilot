# 三角形的内心与内切圆

## Core Idea
与三角形各边都相切的圆叫做三角形的**内切圆**。
内切圆的圆心叫做三角形的**内心**。
**内心的性质**：
1. **位置**：内心是三角形**三个内角平分线**的交点。
2. **距离**：内心到三角形三边的距离相等（都等于内切圆的半径 $r$）。
3. **面积公式**：设 $\triangle ABC$ 的三边长分别为 $a, b, c$，周长为 $L = a+b+c$，内切圆半径为 $r$，则三角形的面积 $S = \frac{1}{2}Lr = \frac{1}{2}(a+b+c)r$。

## Learning Steps
1. **理解概念**：对比外接圆与外心，准确掌握内切圆和内心的定义。
2. **掌握内心的确定**：知道内心是通过作角的平分线得到的，并理解为什么角平分线的交点到三边的距离相等。
3. **应用面积法求半径**：掌握用 $S = \frac{1}{2}(a+b+c)r$ 这个重要公式去求内切圆的半径，这是初中几何非常常用的“面积等量代换”法。
4. **直角三角形特例**：熟记直角三角形内切圆半径的快捷公式 $r = \frac{a+b-c}{2}$（其中 $a,b$ 为直角边，$c$ 为斜边）。

## Common Pitfalls
- **内心与外心混淆**：
  - 外心：三边垂直平分线交点，到三个顶点距离相等，外接圆的圆心。
  - 内心：三个内角平分线交点，到三条边距离相等，内切圆的圆心。
- 忘记直角三角形内切圆半径公式，或者在普通三角形中乱用该快捷公式。
- 利用面积公式时，常常忘记乘 $\frac{1}{2}$。

## Parent Guidance
- 让孩子把“内心-内角平分线-到边距离等”、“外心-外边垂直平分线-到点距离等”编成口诀记住，并在纸上亲自作图验证。
- 可以出一道简单的直角三角形题（如边长为3,4,5），让孩子分别用面积法和快捷公式求一次内切圆半径，验证结果是否相同，增加学习成就感。
- 强调“等面积法”不仅可以求高，还可以求内切圆半径。

## Linked Mistake Concepts
- 三角形面积的多种计算方式运用不熟练
- 角平分线的性质定理遗忘
- 切线长定理在求内切圆问题中的缺失（直角三角形快捷公式的推导就是靠切线长定理）

## Examples

**Example 1**
已知直角 $\triangle ABC$ 的两直角边 $a = 6$，$b = 8$，求其内切圆的半径 $r$。

**Solution**
```text
首先根据勾股定理求出斜边 c 的长：
c = √(a² + b²) = √(6² + 8²) = 10。

方法一：使用面积法
三角形的面积 S = 1/2 * a * b = 1/2 * 6 * 8 = 24。
又因为 S = 1/2 * (a + b + c) * r，
即 24 = 1/2 * (6 + 8 + 10) * r
24 = 12 * r
r = 2。

方法二：使用直角三角形内切圆半径公式
r = (a + b - c) / 2 = (6 + 8 - 10) / 2 = 4 / 2 = 2。
答：该直角三角形内切圆的半径为 2。
```

**Example 2**
已知点 $I$ 是 $\triangle ABC$ 的内心，$\angle A = 60^\circ$，求 $\angle BIC$ 的度数。

**Solution**
```text
因为 I 是 △ABC 的内心，所以 BI、CI 分别平分 ∠ABC 和 ∠ACB。
所以 ∠IBC + ∠ICB = 1/2(∠ABC + ∠ACB)。
在 △ABC 中，∠ABC + ∠ACB = 180° - ∠A = 180° - 60° = 120°。
所以 ∠IBC + ∠ICB = 1/2 * 120° = 60°。
在 △BIC 中，∠BIC = 180° - (∠IBC + ∠ICB) = 180° - 60° = 120°。
(规律：∠BIC = 90° + 1/2 ∠A)
```
