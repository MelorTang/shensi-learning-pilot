# 整式的加减

## Core Idea
整式的加减实质上就是去括号和合并同类项。
在进行整式加减运算时，一般步骤为：
1. 列出算式，给每个多项式加上括号。
2. 去括号：括号前面是“+”号，去掉括号和“+”号，括号里各项都不变号；括号前面是“-”号，去掉括号和“-”号，括号里各项都要变号。
3. 合并同类项：将多项式中的同类项进行合并，得到最简结果。

整式的加减运算结果仍为整式。

## Learning Steps
1. 掌握去括号法则：通过分配律理解去括号的本质。特别强化负号去括号时的“全变号”规则。
2. 规范列式：在解答“求多项式 A 与 B 的差”的题目时，强制要求先写成 `(A) - (B)` 的形式，然后再去括号。
3. 综合运用：结合升降幂排列、同类项合并，完成复杂的整式加减运算。
4. 应用题建模：将实际问题转化为整式的加减问题（如面积的相减、路程的计算）。

## Common Pitfalls
- 去括号时变号不彻底：当括号前是负号时，学生往往只改变了括号内第一项的符号，后面的项忘记变号。例如 $-(x^2 - 2x + 1)$ 误写为 $-x^2 - 2x + 1$。
- 列式不加括号：题目要求“从 $A$ 减去 $B$”，学生直接写成 $A - B$（若 $B$ 是多项式则直接散写），导致后续运算全错。正确的应该是 $A - (B)$。
- 添括号的错误：在需要把某些项括起来时，如果括号前是负号，括号内的项没有改变符号。

## Parent Guidance
- 重点突击“负号”：负号是初中代数的第一大拦路虎。在孩子做题时，专门盯着他们处理括号前是负号的步骤，要求大声读出“负负得正，负正得负”。
- 培养草稿习惯：整式加减项数多，容易眼花。鼓励孩子在草稿纸上每一步都写清晰，不要跳步，跳步最容易导致符号错误。
- 强调验证化简求值题：做化简求值题时，一定要先化简到最简形式再去代入数值，切忌一开始就带入数值强算，那不仅容易错，且违背了考察初衷。

## Linked Mistake Concepts
- 方程去分母与去括号混淆：解方程去分子括号时忘记整体性，同理整式减法中减式也是一个整体。
- 乘法分配律与去括号混淆：括号前有系数时，如 $-2(x - 3)$，既要乘系数又要变号，学生常写成 $-2x - 3$ 或 $-2x - 6$。

## Examples
```text
Question:
已知 A = 2x^2 - 3x + 1，B = x^2 + 5x - 4。求 A - 2B，并将结果按 x 降幂排列。

Solution:
A - 2B
= (2x^2 - 3x + 1) - 2(x^2 + 5x - 4)    // 注意给 B 加上括号并乘以 2
= 2x^2 - 3x + 1 - 2x^2 - 10x + 8       // 去括号，注意 -2 乘以 5x 和 -4 时的变号
= (2x^2 - 2x^2) + (-3x - 10x) + (1 + 8) // 合并同类项
= -13x + 9

按 x 降幂排列已经是 -13x + 9，因为 x 的一次项在前面，常数项在后面。
最终答案为：-13x + 9。
```
