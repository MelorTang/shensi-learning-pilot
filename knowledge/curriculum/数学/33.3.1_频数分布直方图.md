# 频数分布直方图

## Core Idea
为了直观地展示数据的分布规律（哪些数据多，哪些数据少），我们通常将数据分组，并绘制**频数分布直方图**。
- **频数**：落在各个小组内的数据的个数。
- **组距**：每个小组的两个端点之间的距离（通常各组组距相等）。
- **组数**：将数据分成的组的个数。$\text{组数} = \frac{\text{最大值} - \text{最小值}}{\text{组距}}$ （结果进一取整）。
- **直方图的特征**：横轴表示数据分组情况，纵轴表示频数。各个矩形是**紧密相连**的（与条形图分离的矩形不同），矩形的高度代表了该组的频数。

直方图能让人一眼看出数据的集中趋势和波动范围。

## Learning Steps
1. **计算极差**：找出数据中的最大值和最小值，求差。
2. **决定组距和组数**：根据极差和数据量，合理选择组距，并计算出组数。
3. **列频数分布表**：确定各组的边界（为了避免数据落在线上，通常边界多取一位小数，例如 $49.5 \sim 59.5$），统计落在每组的数据个数（画正字法）。
4. **画直方图**：标好横纵坐标及其单位，画出紧密相连的矩形，并标上相关数据。
5. **读图分析**：能从已有的直方图中提取总人数、某组的频数、占比，甚至估算平均数和中位数落在哪一组。

## Common Pitfalls
- **直方图与条形统计图混淆**：条形图的柱子是分开的，直方图的柱子必须是紧挨着的（因为数据是连续分组的）。
- **边界值归属不清**：如果分组是 $50\sim60$、$60\sim70$，不清楚数据 $60$ 算在哪一组。通常规定“左闭右开”（包含左边数字，不包含右边数字）。
- **组数计算不进位**：例如算出 $\frac{\text{最大值}-\text{最小值}}{\text{组距}} = 4.2$，组数应该是 $5$ 组，而不是四舍五入的 $4$ 组。
- **漏数、多数**：在数原数据画“正”字时不够细心，导致频数统计错误。

## Parent Guidance
- 让孩子把数据分组的过程看作是“把豆子分拣到不同大小的盒子里”。告诉孩子每个盒子（组距）必须一样宽，这样比较才公平。
- 如果孩子在数频数时容易错，教孩子一种习惯：每统计完一个数字，就在原数据上划一笔划掉，保证不重不漏。
- 引导孩子看图说话：“从这个图上，你一眼看过去，哪一部分的人最多？最差的有多少人？”培养数据直觉。

## Linked Mistake Concepts
- 频率与频数的关系（$\text{频率} = \frac{\text{频数}}{\text{总数}}$）掌握不好
- 组中值的计算公式不熟练（$\frac{\text{上限}+\text{下限}}{2}$）
- 数轴区间的表示法及开闭区间概念模糊

## Examples

**Example 1**
一组数据的最大值是 $98$，最小值是 $45$。若取组距为 $10$，则这组数据应该分成几组？

**Solution**
```text
极差 = 最大值 - 最小值 = 98 - 45 = 53。
组数 = 极差 / 组距 = 53 / 10 = 5.3。
因为组数必须是整数，且不能漏掉任何数据，所以只要有余数就必须进一。
5.3 进一取整，得到 6。
答：这组数据应该分成 6 组。
```

**Example 2**
在某班级 $40$ 名学生的身高频数分布直方图中，发现 $160\text{ cm} \sim 165\text{ cm}$ 这一组的矩形高度是最高的。已知整个直方图中，只有 $4$ 组，频数之比从左到右依次是 $1:2:4:1$。求最高这一组的频数是多少？

**Solution**
```text
设这 4 组的频数分别为 x, 2x, 4x, x。
因为总人数是 40 人，
所以 x + 2x + 4x + x = 40
8x = 40
x = 5。
最高的一组占的比例最大，即 4x 这组。
频数 = 4 * 5 = 20。
答：最高这一组的频数是 20 人。
```
