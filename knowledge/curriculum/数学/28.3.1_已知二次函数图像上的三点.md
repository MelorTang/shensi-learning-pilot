# 已知二次函数图像上的三点

## Core Idea
当已知二次函数图像上的任意三个不共线的点时，最通用的方法是设函数的一般式 $y=ax^2+bx+c$，通过待定系数法列出一个三元一次方程组，求出 $a, b, c$ 的值。

## Learning Steps
1. 设出二次函数的一般形式 $y=ax^2+bx+c$ ($a \neq 0$)。
2. 将已知的三个点的坐标分别代入解析式，得到关于 $a, b, c$ 的三元一次方程组。
3. 掌握解三元一次方程组的方法（通常先消去 $c$ 比较容易）。
4. 将求得的系数代回原设的解析式中。

## Common Pitfalls
- 解三元一次方程组时计算错误，尤其是在消元过程中的符号问题。
- 如果三个点中有与 $y$ 轴的交点 $(0, c)$，没有优先利用，导致计算复杂化。

## Parent Guidance
家长可以检查孩子在解三元一次方程组时的书写习惯，提醒他们对等式进行编号（如①②③），并清晰写出消元的过程（如②-①得到④）。这有助于提高准确率，也有利于检查错误。

## Linked Mistake Concepts
- 如果已知的三个点中有两个点的 $y$ 坐标相同，这其实意味着可以通过这两个点找到对称轴。学生往往忽略这一特殊情况，仍然机械地解三元方程组。

## Examples
```text
题目：
已知二次函数的图像经过 A(-1, 0)、B(0, -3)、C(2, 5) 三点，求这个二次函数的解析式。

解答：
设该二次函数的解析式为 y = ax^2 + bx + c (a ≠ 0)。
因为图像经过点 B(0, -3)，所以 c = -3。
将 c = -3 代入，得 y = ax^2 + bx - 3。
再将点 A(-1, 0) 和 C(2, 5) 的坐标代入得：
0 = a(-1)^2 + b(-1) - 3  =>  a - b = 3  (①)
5 = a(2)^2 + b(2) - 3    =>  4a + 2b = 8  => 2a + b = 4  (②)

将 ① 和 ② 联立解方程组：
由①得 a = b + 3，代入②：
2(b + 3) + b = 4
2b + 6 + b = 4
3b = -2
b = -2/3
代回 a = b + 3：
a = -2/3 + 3 = 7/3

所以该二次函数的解析式为：y = (7/3)x^2 - (2/3)x - 3。
```
