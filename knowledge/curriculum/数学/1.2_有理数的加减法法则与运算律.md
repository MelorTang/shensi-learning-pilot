# 有理数的加减法法则与运算律

## Core Idea

有理数的加法和减法是将正负数合并的运算。加法的核心是“同号求和，异号求差”；减法的核心是“减去一个数，等于加上这个数的相反数”，将减法统一转化为加法。运算律可用于简化计算。

## Learning Steps

- **加法法则**：
  1. 同号两数相加：取相同的符号，并把绝对值相加。
  2. 异号两数相加：绝对值相等时和为0；绝对值不等时，取绝对值较大的加数的符号，并用较大的绝对值减去较小的绝对值。
- **减法法则**：把减号变成加号，同时把减数变成它的相反数（即“变号加”）。
- **运算律应用**：加法交换律和结合律。将同号数、凑整的数或互为相反数的数先结合计算。

## Parent Guidance
让孩子背诵口诀并实际操作：“减去一个数等于加上它的相反数”，监督孩子在草稿纸上**必须写出化为加法的那一步**，不要心算跨步。当孩子异号相加出错时，问他：“正负两军打仗，哪边人多？最后剩下的是哪边的人？”

## Common Pitfalls

- **异号相加定错符号**：比如 -8 + 3，算出了绝对值的差5，但符号忘记跟着绝对值大的一方走，写成了 5。
- **减法未变号**：执行减法时，只变了减号没变减数符号，比如 5 - (-3) 算成了 5 + (-3) = 2。
- **去括号符号错误**：在简便运算中，如 - ( -4 + 3 ) 展开时，括号里只改了第一个数字的符号。

## Linked Mistake Concepts
- 异号相加符号错误
- 减法化加法错误
- 去括号变号错误

## Examples

### Example 1

题目：
计算：(-12) - (-8) + (-5)

解法：

```text
第一步，将减法化为加法：(-12) + (+8) + (-5)
第二步，利用加法交换律将同号放在一起：[(-12) + (-5)] + 8
第三步，计算：(-17) + 8 = -9。
```
