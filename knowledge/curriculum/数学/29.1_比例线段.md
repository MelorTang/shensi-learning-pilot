# 比例线段

## Core Idea
比例线段是指在两条线段（或多条线段）中，如果其中两条线段的长度之比等于另外两条线段的长度之比，那么这四条线段就叫做成比例线段。这是相似几何图形研究的基础，也是连接代数比例关系和几何图形的重要桥梁。

## Learning Steps
1. 复习小学学过的比例的基本性质（内项之积等于外项之积）。
2. 理解线段的比：两条线段长度的比值。注意单位必须统一。
3. 学习成比例线段的概念，掌握黄金分割比例等特殊比例线段。
4. 学习平行线分线段成比例定理，了解其推论。

## Common Pitfalls
- 求两条线段的比时，未统一单位。例如一条线段长 2m，另一条长 50cm，直接写成 2:50，导致错误。
- 书写比例式时，线段的对应关系错位。如将 $AB:CD = EF:GH$ 错误写成 $AB:EF = GH:CD$。

## Parent Guidance
在日常生活中，可以和孩子探讨地图上的比例尺、照片的放大与缩小等，让孩子直观感受线段按比例缩放的实际应用。

## Linked Mistake Concepts
- 比例的合比性质与等比性质运用不熟练，在推导复杂的几何比例式时容易出现代数变形错误。
- 误认为两条线段的比与它们所在的直线位置有关，实际上线段的比仅仅是它们长度的比值。

## Examples
```text
题目：
已知线段 a = 2 cm, b = 30 mm, c = 4 cm，若 a, b, c, d 四条线段成比例，求线段 d 的长度。

解答：
首先统一单位，将所有的线段长度单位转换为 cm。
a = 2 cm
b = 30 mm = 3 cm
c = 4 cm

因为 a, b, c, d 四条线段成比例，即 a : b = c : d。
代入数值：
2 : 3 = 4 : d
根据比例的基本性质（内项之积等于外项之积）：
2 * d = 3 * 4
2d = 12
d = 6

所以，线段 d 的长度为 6 cm。
```
