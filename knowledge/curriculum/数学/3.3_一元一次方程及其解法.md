# 一元一次方程及其解法

## Core Idea (核心概念)
一元一次方程是指只含有一个未知数（元），并且未知数的次数都是1（次）的整式方程。解一元一次方程的核心思想是“化归”，即通过运用等式的基本性质，对含有未知数的方程进行变形，最终化为 $x = a$ 的形式。

## Learning Steps (学习步骤)
1. **识别一元一次方程**：掌握一元一次方程的三个条件（只含一个未知数、未知数最高次数为1、是整式方程）。
2. **掌握等式基本性质**：
   - 性质1：等式两边同时加（或减）同一个数或式子，等式仍然成立。
   - 性质2：等式两边同时乘（或除以）同一个不为零的数，等式仍然成立。
3. **熟练掌握解方程的基本步骤**：
   - 去分母（方程两边同乘各分母的最小公倍数）。
   - 去括号（注意括号前的符号和分配律的正确运用）。
   - 移项（把含有未知数的项移到方程一边，常数项移到另一边，**移项要变号**）。
   - 合并同类项（把方程化为 $ax = b$ 的形式）。
   - 系数化为1（方程两边同除以 $a$）。

## Common Pitfalls (常见易错点)
- **移项不变号**：移动项到等号另一侧时忘记改变正负号。
- **去括号漏乘或符号错误**：括号外有系数时，没有乘到括号内的每一项；括号前面是负号时，括号里的项没有全部变号。
- **去分母漏乘常数项**：方程两边同乘最小公倍数时，不含分母的常数项忘记乘。

## Parent Guidance
家长在检查孩子的作业时，要重点关注孩子解方程的“过程”而非仅仅是“结果”。如果孩子算错了，不要直接告诉答案，而是让孩子倒推检查：先看最后一步“系数化为1”有没有算错；再看前一步“合并同类项”是否正确；接着检查“移项”时有没有变号；然后再看“去括号”时负号分配对不对；最后看“去分母”时是否漏乘。通过这种“复盘”方法，帮助孩子养成严谨的数学演算习惯。

## Linked Mistake Concepts
- 【整式的加减错误】在合并同类项时，对系数的加减运算（特别是有理数的加减）不熟练。
- 【分数运算错误】在去分母时找错最小公倍数，或者在系数化为1时除法转化为乘法的倒数运算出错。

## Examples (例题)

**例题 1**：解方程：$2(x - 3) = 3x + 4$

```text
1. 去括号，得：
   2x - 6 = 3x + 4
2. 移项（将3x移到左边，-6移到右边），得：
   2x - 3x = 4 + 6
3. 合并同类项，得：
   -x = 10
4. 系数化为1（两边同除以-1），得：
   x = -10
```

**例题 2**：解方程：$\frac{2x - 1}{3} - \frac{x + 2}{4} = 1$

```text
1. 去分母（方程两边同乘12，注意常数项1也要乘12），得：
   4(2x - 1) - 3(x + 2) = 12
2. 去括号（注意第二个括号前是负号，里面要变号），得：
   8x - 4 - 3x - 6 = 12
3. 移项，得：
   8x - 3x = 12 + 4 + 6
4. 合并同类项，得：
   5x = 22
5. 系数化为1，得：
   x = 22/5 （或 4.4）
```
