# 已知二次函数图像的顶点或对称轴

## Core Idea
当题干给出了二次函数的顶点坐标 $(h, k)$，或者给出了对称轴 $x=h$ 及最值时，使用顶点式 $y=a(x-h)^2+k$ 是最快捷的方法。这样可以立刻减少两个未知数，只需再代入另一个点即可求出 $a$。

## Learning Steps
1. 识别题目中的隐含信息，如“最值”、“最高点”、“最低点”、“对称轴”等，这些都指向顶点。
2. 根据给定的顶点坐标 $(h, k)$，直接设解析式为 $y=a(x-h)^2+k$。
3. 寻找题目中给出的另外一个非顶点坐标的点。
4. 将该点坐标代入，求解 $a$。

## Common Pitfalls
- 设顶点式时忽略了首项系数 $a$，直接写成 $y=(x-h)^2+k$，导致得到的函数开口大小或方向不符。
- 坐标符号提取错误，如已知顶点为 $(-2, 3)$，误设为 $y=a(x-2)^2+3$（正确应为 $(x+2)^2$）。

## Parent Guidance
提醒孩子遇到“最大值”、“最小值”、“顶点”、“对称轴”这些字眼时，要有条件反射——想到顶点式。可以和孩子一起玩“找关键词”的游戏，提高他们对题干信息的敏感度。

## Linked Mistake Concepts
- 顶点坐标的理解：有些题目给出“当 $x=2$ 时，有最小值为 $-1$”，学生不能立刻反应出这就是顶点坐标 $(2, -1)$。
- 在对称轴上的点不止一个，但只有顶点决定了函数的极值，需要注意区分。

## Examples
```text
题目：
已知一个二次函数的对称轴为直线 x = 1，图像经过原点 (0,0) 和点 (3, -3)，求这个二次函数的解析式。

解答：
已知对称轴为 x = 1，可以设其解析式为：
y = a(x - 1)^2 + k (a ≠ 0)

将 (0,0) 和 (3, -3) 分别代入得到：
0 = a(0 - 1)^2 + k  => a + k = 0 (①)
-3 = a(3 - 1)^2 + k => 4a + k = -3 (②)

用②减去①：
(4a + k) - (a + k) = -3 - 0
3a = -3
a = -1

将 a = -1 代入①：
-1 + k = 0
k = 1

所以该二次函数的解析式为：
y = -(x - 1)^2 + 1
展开得（可选）：
y = -x^2 + 2x
```
