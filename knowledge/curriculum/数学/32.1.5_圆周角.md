# 圆周角

## Core Idea
**圆周角**是指顶点在圆上，并且两边都与圆相交的角。
**核心定理**：
1. **圆周角定理**：在同圆或等圆中，同弧或等弧所对的圆周角相等，都等于这条弧所对的圆心角的一半。
2. **推论 1**：半圆（或直径）所对的圆周角是直角（$90^\circ$）；$90^\circ$的圆周角所对的弦是直径。
这一定理是沟通角度之间联系的重要桥梁，能把圆上的角化归为圆心角的计算。

## Learning Steps
1. **识别圆周角**：满足两个条件：顶点在圆上；两边和圆相交。缺一不可。
2. **理解定理证明**：掌握通过作直径将圆周角转化为圆心角的过程（利用等腰三角形外角性质）。
3. **熟练运用定理**：遇到“求角”问题，优先寻找“同弧”所对的圆周角，看是否能建立等量关系。
4. **记住特殊结论**：直径所对的圆周角为直角。在解题中看到直径，就要想到直角；需要构造直角，就要联想连直径。

## Common Pitfalls
- 忽略了前提条件“**同弧或等弧**”。如果没有这个前提，圆周角之间没有必然的大小关系。
- 顶点不在圆上的角（例如圆内角或圆外角）错当成圆周角去计算。
- 角度换算时，忘记“**一半**”的关系，把圆周角直接等于圆心角。

## Parent Guidance
- 可以问孩子：“如果在圆周上随便取一个点和一条固定的弧相连，角度会变吗？”让孩子通过画图感知，只要弧不变，不管顶点跑去哪，角度都不变（同侧），这就像是一个“视线恒定”的照相机。
- 当题目里出现“直径”二字时，提醒孩子：“第一反应是什么？” 培养孩子看到直径就自动去找直角，看到直角就想直径的反射弧。
- 让孩子检查自己画的角，顶点到底是不是正好落在圆周那根曲线上。

## Linked Mistake Concepts
- 等腰三角形的性质及外角定理掌握薄弱
- 圆内角、圆外角与圆周角的混淆
- 对“弧所对的角”空间想象能力不足

## Examples

**Example 1**
已知在 $\odot O$ 中，圆心角 $\angle AOB = 100^\circ$，点 $C$ 在优弧 $AB$ 上。求圆周角 $\angle ACB$ 的度数。

**Solution**
```text
因为 ∠AOB 和 ∠ACB 是同弧（劣弧 AB）所对的圆心角和圆周角，
根据圆周角定理：同弧所对的圆周角等于它所对圆心角的一半，
所以 ∠ACB = 1/2 * ∠AOB = 1/2 * 100° = 50°。
```

**Example 2**
在 $\odot O$ 中，$AB$ 是直径，点 $C$ 在圆上（不与 $A, B$ 重合），若 $\angle A = 30^\circ$，求 $\angle B$ 的度数。

**Solution**
```text
因为 AB 是 ⊙O 的直径，点 C 在圆上，
根据推论：直径所对的圆周角是直角，
所以 ∠ACB = 90°，即 △ABC 是直角三角形。
在 Rt△ABC 中，∠A + ∠B = 90°。
因为 ∠A = 30°，
所以 ∠B = 90° - 30° = 60°。
```
