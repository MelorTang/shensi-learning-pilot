# 三角形全等的判定

## Core Idea
全等三角形的判定定理旨在用最少的条件（通常是三个）来确定两个三角形全等。主要的判定方法有：SSS（边边边）、SAS（边角边）、ASA（角边角）、AAS（角角边），以及直角三角形特有的HL（斜边直角边）。这是几何推理证明的重中之重。

## Learning Steps
1. **探索并掌握判定定理**：通过尺规作图验证 SSS, SAS, ASA, AAS，并理解为什么 SSA (边边角) 和 AAA (角角角) 不能判定三角形全等。
2. **书写证明过程**：熟练掌握“准备条件 -> 凑齐三个条件大括号 -> 得出全等结论 -> 得出边角相等”的标准证明格式。
3. **挖掘隐含条件**：在复杂图形中，主动寻找公共边、公共角、对顶角等隐含的相等条件。
4. **直角三角形的HL定理**：明确HL定理仅适用于直角三角形，即斜边和一条直角边对应相等。

## Common Pitfalls
- **错误使用SSA或AAA**：这是最致命的错误，看到两边一角相等就直接判全等，而不看这个角是不是两边的“夹角”。
- **证明格式极度不规范**：大括号内条件顺序不按定理的字母顺序写，或者缺少前置的推导步骤。
- **张冠李戴**：把题目给的一个三角形内的条件，硬塞给另一个三角形来证明全等。

## Parent Guidance
这是几何证明的第一座高山。家长应严格要求孩子写出规范的“三段论”大括号格式，不可偷工减料。可以用口诀提醒孩子：“找全等，看条件，边角边，角边角，边边边，角角边，直角三角形加HL；千万别用边边角和角角角。”遇到难题时，引导孩子把两个重叠的三角形用彩笔描出来，或者在草稿纸上分开画。

## Linked Mistake Concepts
- 尺规作图（判断定理的本质其实就是唯一作图法）
- 三角形的有关概念（如角平分线、中线、高，它们能提供边角相等的已知条件）

## Examples
题目：如图，在△ABC和△DCB中，已知 AB=DC，AC=DB。求证：∠A = ∠D。
```text
证明：
在 △ABC 和 △DCB 中：
{
  AB = DC (已知)
  AC = DB (已知)
  BC = CB (公共边)
}
所以 △ABC ≌ △DCB (SSS)。
所以 ∠A = ∠D (全等三角形的对应角相等)。
```
