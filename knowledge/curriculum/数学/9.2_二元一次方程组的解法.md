# 二元一次方程组的解法

## Core Idea
掌握代入消元法和加减消元法，体会解多元方程组的核心思想——“消元”，即将二元化为一元，再转化为一元一次方程来求解。

## Learning Steps
1. 学习代入消元法：从一个方程中解出一个未知数（用含另一个未知数的代数式表示），再代入另一个方程。
2. 学习加减消元法：通过对方程两边同乘某个数，使得某个未知数的系数互为相反数或相等，然后相加或相减。
3. 比较两种方法，根据方程的具体特征灵活选择最优解法。

## Common Pitfalls
- 加减消元时，同号相减、异号相加的符号极易出错，经常忘记等号右边的常数项也要相加减。
- 代入消元时忘记加括号，导致乘法分配律没有正确展开。

## Parent Guidance
监督孩子在解出方程组的答案后养成检验的好习惯，即把求出的解重新代入原方程组中验证左右两边是否相等。初学时要强调书写的规范性，如标注方程①、方程②，避免思路混乱。

## Linked Mistake Concepts
- 在用加减消元法乘以系数时，漏乘了方程中某些项（尤其是常数项）。
- 解出一个未知数后，代入求解另一个未知数时计算出错。

## Examples
题目：
用加减消元法解二元一次方程组：
{ 3x + 2y = 7   ①
{ 2x - 2y = 3   ②

解答：
```text
观察方程组，发现 y 的系数互为相反数，可直接使用加法消元。
① + ② 得：
(3x + 2x) + (2y - 2y) = 7 + 3
5x = 10
解得 x = 2

将 x = 2 代入 ① 中：
3(2) + 2y = 7
6 + 2y = 7
2y = 1
解得 y = 0.5

所以方程组的解为：
{ x = 2
{ y = 0.5
```
