# 圆锥及其侧面展开图

## Core Idea
了解圆锥的特征，掌握圆锥的侧面展开图是一个扇形。理解扇形的弧长即底面圆的周长，扇形的半径即圆锥的母线长。

## Learning Steps
1. 认识圆锥体的各部分名称：底面、侧面、顶点、高以及母线。
2. 研究圆锥的侧面展开图，明确扇形与圆锥之间的对应量（弧长=底面周长，扇形半径=母线）。
3. 掌握圆锥侧面积公式 $S=\pi r l$ 及表面积的计算公式。

## Common Pitfalls
- 把圆锥的高当作侧面展开图扇形的半径，混淆母线与高的概念。
- 使用侧面积公式时错误地将母线 $l$ 替换成圆锥的高 $h$。

## Parent Guidance
和孩子一起用纸卷一个圆锥帽，然后将其展开，帮助他们理解母线、底面周长和扇形参数之间的对应关系。提醒孩子在遇到计算问题时，可以先画出一个直角三角形来找半径、高和母线的关系（勾股定理）。

## Linked Mistake Concepts
- 在计算表面积时，常常忽略加上底面圆的面积。
- 不清楚勾股定理在圆锥底面半径、高、母线三者构成的直角三角形中的应用。

## Examples
题目：
已知一个圆锥的底面半径为 $3\text{ cm}$，高为 $4\text{ cm}$。求它的母线长和侧面积。

解答：
```text
利用勾股定理求母线长 l：
l = √(r² + h²) = √(3² + 4²) = √25 = 5 cm。
圆锥侧面积 S_侧 = π * r * l = π * 3 * 5 = 15π cm²。
```
