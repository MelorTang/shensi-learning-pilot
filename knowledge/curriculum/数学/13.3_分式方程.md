# 分式方程

## Core Idea
分式方程的定义：分母中**含有未知数**的方程叫做分式方程。
解分式方程的核心思想是**“转化”**：将分式方程转化为我们熟悉的一元一次方程或一元二次方程（即整式方程）。
基本步骤为：两边同乘最简公分母，去掉分母，转化为整式方程，解得未知数的值。
**最重要的一步是检验**：因为在去分母时，同乘的最简公分母有可能等于 $0$（而等式两边同乘 $0$ 是没有意义的，会导致产生增根）。所以解出整式方程的根后，**必须**将其代入原方程的最简公分母进行检验，如果使公分母为 $0$，这个根就是增根，必须舍去。

## Learning Steps
1. 找最简公分母：对所有分母进行因式分解，找出涵盖所有特征因式的最简公分母。
2. 去分母：方程两边每一项（包括常数项）都要乘以最简公分母，化为整式方程。
3. 解整式方程：通过去括号、移项、合并同类项解出未知数。
4. 检验（绝对不能少）：把解得的未知数代入第一步找出的最简公分母中。
   - 若值不为 $0$，则是原方程的解。
   - 若值为 $0$，则是增根，原分式方程无此解（若只有这一个解则方程无解）。
5. 写出结论：清晰写明“经检验，x = ... 是（或不是）原方程的解”。

## Common Pitfalls
- 漏乘常数项：例如方程 $\frac{2x}{x-1} - 1 = \frac{3}{x-1}$，两边同乘 $x-1$ 时，常数项 $1$ 忘记乘 $x-1$，错写成 $2x - 1 = 3$。
- 不打括号导致符号错误：去分母后，如果分子是一个多项式且前面带有负号，去分母后没有把这个多项式用括号括起来，导致减号没有分配给后面的项。
- 忘记检验：费了半天劲解出了方程，却忘记了最后一步检验，结果包含增根被扣掉全部分数。分式方程不检验等于没做。

## Parent Guidance
- 设立“检验法庭”：告诉孩子，解整式方程就像抓嫌疑人，抓到了（得出 $x$）不能直接判刑，必须送上“检验法庭”（代入公分母）。如果法庭判定他把分母炸成了零，这人就是个冒牌货（增根），必须当庭释放（舍去）。
- 检查漏乘：在孩子做完去分母那一步后，马上让他数一数原方程有几项，去完分母后是不是还有这几项，并且每一项是不是都乖乖乘了公分母，特别盯紧那些光秃秃的常数项。

## Linked Mistake Concepts
- 化简与解方程的混用：上面提到过，在分式化简（加减法）时去分母。这里再强调一遍：有等号的叫方程，为了求 $x$，可以扔掉分母；没有等号的叫化简，只是换个造型，分母绝不能扔。

## Examples
```text
Question:
解方程：\frac{3}{x-2} - \frac{1}{x} = \frac{2}{x(x-2)}

Solution:
第一步：确定最简公分母
分母分别为：x-2，x，x(x-2)
所以最简公分母是：x(x-2)。

第二步：方程两边同乘 x(x-2)，去分母
3x - 1(x-2) = 2
// 注意：负号后面的 x-2 一定要加括号！

第三步：解这个整式方程
3x - x + 2 = 2
2x = 0
x = 0

第四步：检验（核心步骤！）
把 x = 0 代入最简公分母 x(x-2) 中，
当 x = 0 时，x(x-2) = 0 \times (-2) = 0。
说明公分母为 0，所以 x=0 是增根。

第五步：结论
经检验，x=0 是原方程的增根。
所以，原分式方程无解。
```
