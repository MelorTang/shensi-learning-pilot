# 圆柱及其侧面展开图

## Core Idea
理解圆柱的形成及其结构特征，掌握圆柱侧面展开图是一个矩形，且其长等于底面圆的周长，宽等于圆柱的高。

## Learning Steps
1. 认识圆柱体，掌握圆柱的底面、侧面和高的定义。
2. 探究圆柱侧面展开图，理解矩形的长与底面周长的对应关系。
3. 利用圆的周长和面积公式，计算圆柱的侧面积和表面积。

## Common Pitfalls
- 计算侧面积时，忘记底面圆的周长公式，漏乘 $\pi$ 或 2。
- 计算圆柱的表面积时，容易只算一个底面，或者只算侧面积。

## Parent Guidance
拿一个易拉罐或薯片筒，去掉上下底面后剪开，让孩子直观地看到圆柱的侧面展开图是矩形。同时可以让他们测量展开后矩形的长，并与底面圆的周长进行对比验证。

## Linked Mistake Concepts
- 混淆圆柱体积和表面积的公式。
- 对“侧面展开图的长就是底面周长”这一核心对应关系理解不到位，导致无法通过展开图反推圆柱的底面半径。

## Examples
题目：
已知一个圆柱的底面半径为 $4\text{ cm}$，高为 $10\text{ cm}$。求它的侧面积和全面积（结果保留 $\pi$）。

解答：
```text
底面周长 C = 2 * π * 4 = 8π cm。
侧面积 S_侧 = C * h = 8π * 10 = 80π cm²。
底面积 S_底 = π * r^2 = π * 4^2 = 16π cm²。
全面积 S_全 = S_侧 + 2 * S_底 = 80π + 32π = 112π cm²。
```
