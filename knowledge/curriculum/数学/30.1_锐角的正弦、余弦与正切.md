# 锐角的正弦、余弦与正切

## Core Idea
锐角的三角比（正弦、余弦、正切）是直角三角形中边长之比的抽象化。只要锐角的度数确定，这些比值就是固定不变的，与直角三角形的大小无关。这是从几何过渡到三角学的核心概念。

## Learning Steps
1. 复习相似三角形性质，理解“对于同一个锐角，无论所在的直角三角形多大，其对应边的比值恒定”。
2. 引入三角函数的定义：对边与斜边之比（正弦），邻边与斜边之比（余弦），对边与邻边之比（正切）。
3. 学习书写规范（如 $\sin A$, $\cos A$, $\tan A$）及读法。
4. 认识三角比的取值范围，体会角的变化对三角比值的影响。

## Common Pitfalls
- 混淆“对边”和“邻边”。特别是在直角三角形摆放方向不是标准水平/垂直时，容易找错角所对的边。
- 将 $\sin A$ 误认为是 $\sin$ 乘以 $A$。必须强调这是一个不可分割的整体符号。

## Parent Guidance
可以引导孩子观察楼梯的倾斜度，向孩子解释“对边（高度）”和“邻边（水平跨度）”的比值（正切值）就是楼梯的坡度，让抽象的三角比变得具象化。

## Linked Mistake Concepts
- 锐角与直角的混淆：在应用公式时，必须确保三角形是直角三角形，且 $A$ 或 $B$ 是锐角，不能对直角求正弦或余弦。
- 分母为零的误解：在求正切时，虽然 $\tan A = \frac{\text{对边}}{\text{邻边}}$，但因为是锐角，邻边不可能为 0。

## Examples
```text
题目：
在直角三角形 ABC 中，∠C = 90°，AB = 5，BC = 3。求 ∠A 的正弦、余弦和正切值。

解答：
根据勾股定理，求出另一条直角边 AC 的长度：
AC^2 + BC^2 = AB^2
AC^2 + 3^2 = 5^2
AC^2 = 25 - 9 = 16
AC = 4

找准 ∠A 的对边、邻边和斜边：
∠A 的对边是 BC = 3
∠A 的邻边是 AC = 4
斜边是 AB = 5

根据定义：
sin A = 对边 / 斜边 = BC / AB = 3/5
cos A = 邻边 / 斜边 = AC / AB = 4/5
tan A = 对边 / 邻边 = BC / AC = 3/4
```
