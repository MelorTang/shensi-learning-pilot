# 二次三项式的因式分解

## Core Idea
除了使用完全平方公式处理特殊的二次三项式外，对于一般的二次三项式 $ax^2 + bx + c$ 进行因式分解，最常用的方法是**十字相乘法**。
十字相乘法的原理：$(x+p)(x+q) = x^2 + (p+q)x + pq$。
逆向来看，要将 $x^2 + px + q$ 分解因式，核心是找到两个数，使得它们的**乘积等于常数项 $q$**，而它们的**代数和等于一次项系数 $p$**。
对于 $x^2$ 系数不为 1 的 $ax^2 + bx + c$，需要将二次项拆分成 $a_1 x \cdot a_2 x$，将常数项拆分成 $c_1 \cdot c_2$，使得交叉相乘再相加的结果等于一次项 $bx$，即 $a_1 c_2 + a_2 c_1 = b$。

## Learning Steps
1. 标准化：确保多项式是按字母降幂排列的 $ax^2 + bx + c$ 形式，若有公因式必须先提取公因式。
2. 试凑常数项：对于 $x^2 + px + q$，列出常数项 $q$ 的所有可能整数因数对。
3. 验证一次项：在找出的因数对中，筛选出“和等于一次项系数 $p$”的那一对数。
4. 横写结果：将找到的两个数 $m, n$ 直接写成 $(x+m)(x+n)$。如果是 $a \neq 1$ 的十字相乘，则横向读取十字架两端的式子。

## Common Pitfalls
- 符号配对错误：当常数项为正时，两个因数同号（同正或同负，取决于一次项符号）；当常数项为负时，两个因数异号，且绝对值大的数符号与一次项符号相同。学生往往在试凑时搞错正负号。
- 十字相乘横竖读错：在写出十字相乘阵列后，部分学生习惯性交叉读取结果。正确方法是：**交叉相乘（用于验算），横向书写（写最终结果）**。
- 漏提公因式：面对 $3x^2 - 6x - 9$，一上来就尝试复杂的带系数十字相乘，导致过程繁琐且易错，实际上应该先提 $3$。

## Parent Guidance
- 把它当成数独游戏：十字相乘法初期就是凑数字，家长可以把它包装成猜数字游戏：“想两个数，乘起来是 -12，加起来是 1，它们是谁？”多做这种心算，能极大提升十字相乘的速度。
- 总结符号规律口诀：
  “尾正，同号，看中间（常数正，两因数同号，符号和一次项一样）”
  “尾负，异号，大从中间（常数负，两因数异号，绝对值大的那个符号和一次项一样）”。

## Linked Mistake Concepts
- 解方程与因式分解的边界：解一元二次方程时可以用求根公式逆推因式分解，但有时在非实数根或无理数根范围内，题目只要求在有理数范围分解。
- 不能分解误认为自己没凑对：有些二次三项式（如 $x^2+x+1$）在实数范围内无法分解因式，学生会在上面死磕。可以引入判别式 $\Delta = b^2 - 4ac$ 辅助判断，若不是完全平方数，通常不能在整数范围内凑出。

## Examples
```text
Question:
用十字相乘法将下列各式分解因式：
(1) x^2 - 5x + 6
(2) x^2 + 2x - 15
(3) 2x^2 - 7x + 3

Solution:
(1) x^2 - 5x + 6
  // 找两个数，乘积是 +6，和是 -5。
  // 因为乘积为正，和为负，两数必定同为负。
  // 可能的组合：(-1)(-6)和非-7；(-2)(-3)和为-5。
  // 所以两个数为 -2 和 -3。
  原式 = (x - 2)(x - 3)

(2) x^2 + 2x - 15
  // 找两个数，乘积是 -15，和是 +2。
  // 乘积为负，两数一正一负，且正数的绝对值大。
  // 15 = 3 * 5，其中 5 + (-3) = 2。
  // 两个数为 5 和 -3。
  原式 = (x + 5)(x - 3)

(3) 2x^2 - 7x + 3
  // 采用十字相乘阵列
  // 2x^2 拆分为 2x 和 x
  // 常数项 3 拆分为 -1 和 -3 （因为一次项是 -7x，必须全用负数）
  // 阵列：
  // 2x      -1
  //  x      -3
  // 交叉相乘相加：2x*(-3) + x*(-1) = -6x - x = -7x，符合一次项。
  // 横向书写结果：
  原式 = (2x - 1)(x - 3)
```
