# 数轴

## Core Idea

数轴是规定了原点、正方向和单位长度的直线。它将抽象的有理数与直观的几何点建立了一一对应的关系，是数形结合的重要工具。

## Learning Steps

- 掌握数轴的“三要素”：原点（表示0）、正方向（通常向右）、单位长度。
- 能够在数轴上画出表示已知有理数的点。
- 能够读出数轴上的点所表示的有理数。

## Parent Guidance
让孩子亲手画一条标准数轴，检查是否包含了原点、箭头和均匀的刻度。提问：“如果向右是正方向，那么越往左的数是越大还是越小？”帮助孩子建立数字大小与空间位置的联系。

## Common Pitfalls

- **三要素缺失**：画数轴时忘记标出原点、箭头（正方向）或没有统一的刻度（单位长度不一致）。
- **负数方向标错**：在原点左侧标示负数时，距离原点越远，数值的绝对值越大，容易将 -1 和 -2 的位置画反。
- **单位长度不均匀**：不同刻度之间的物理距离不相等。

## Linked Mistake Concepts
- 数轴三要素缺失
- 负数在数轴上的位置错误

## Examples

### Example 1

题目：
请指出在一条向右为正方向的直线上，点A表示 -2.5，点B表示 3。如果它们都在标准的数轴上，它们分别在原点的哪一侧？距离原点多少个单位长度？

解法：

```text
点A在原点左侧，距离原点 2.5 个单位长度。
点B在原点右侧，距离原点 3 个单位长度。
```
