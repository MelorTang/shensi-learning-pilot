# 一元一次不等式

## Core Idea
一元一次不等式是含有一个未知数，且未知数的最高次数为1的不等式。解一元一次不等式的过程与解一元一次方程非常相似，目的都是将不等式化为 x > a 或 x < a 的形式。解集不仅是一个数值，而是一个范围，通常需要在数轴上直观表示出来。

## Learning Steps
1. **理解不等式的解与解集**：知道不等式的解有无数个，所有解组成的集合称为解集。
2. **掌握解一元一次不等式的步骤**：去分母、去括号、移项、合并同类项、系数化为1（重点注意不等号方向）。
3. **在数轴上表示解集**：掌握“大于向右画，小于向左画，有等号画实心圆点，无等号画空心圆圈”的规则。
4. **一元一次不等式的应用**：将实际生活中的不等关系转化为不等式模型并求解。

## Common Pitfalls
- **数轴表示错误**：空心和实心混淆，或者方向画错。
- **去分母漏乘**：方程和不等式一样，去分母时常数项忘记乘公分母。
- **系数化为1时未变号**：当未知数的系数为负数时，化为1时忘记改变不等号方向。
- **应用题中整数解的取舍**：某些实际问题中未知数只能取正整数，但在最终答案中忘记结合实际意义筛选。

## Parent Guidance
在孩子练习解不等式时，家长要着重检查最后一步“系数化为1”。可以和孩子一起玩“找茬”游戏，专门找出题目中忘记变号的错误。对于数轴表示，强调“边界点”的空心实心问题。在应用题上，鼓励孩子多用表格或画图的方式理清数量关系。

## Linked Mistake Concepts
- 一元一次方程的解法（解法步骤完全对应，但系数化为1有区别）
- 数轴的画法与读法（数轴的规范画法是基础）
- 实际问题与代数模型建立（列方程与列不等式在找关系时的区别在于找“>”或“<”的关键虚词）

## Examples
题目：解不等式 \frac{x-1}{2} - 1 \le \frac{2x+1}{3}，并在数轴上表示其解集。
```text
解：
去分母，两边同乘6，得：
3(x - 1) - 6 ≤ 2(2x + 1)
去括号，得：
3x - 3 - 6 ≤ 4x + 2
移项，得：
3x - 4x ≤ 2 + 3 + 6
合并同类项，得：
-x ≤ 11
系数化为1（两边同乘-1，不等号变向），得：
x ≥ -11
在数轴上表示为：在-11处画实心圆点，向右画折线。
```
