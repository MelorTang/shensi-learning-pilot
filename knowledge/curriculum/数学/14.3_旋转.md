# 旋转

## Core Idea
旋转（Rotation）的概念：在平面内，将一个图形绕着一个**定点**沿着某个**方向**转动一个**角度**，这样的图形运动称为旋转。
描述一个旋转需要三个关键要素：
1. **旋转中心**：绕着转动的那个定点。
2. **旋转方向**：顺时针方向或逆时针方向。
3. **旋转角度**：转动的幅度。
旋转的基本性质：
1. **形状和大小不变**：旋转前后的图形全等。
2. 任意一对对应点与旋转中心的连线所成的角都等于**旋转角**。
3. 对应点到旋转中心的**距离相等**。

## Learning Steps
1. 识别旋转现象：如时钟的指针、风车、方向盘的转动。找到旋转中心和判断方向。
2. 明确三大要素：做题时，首先找准谁是中心点，然后看是顺还是逆，最后看转了多少度。这三个要素缺一不可。
3. 寻找对应关系：在旋转后的复杂图形中，找出对应点、对应线段、对应角。特别是“对应点到中心的连线角 = 旋转角”这个性质，是解题的切入点。
4. 画旋转图：利用圆规和量角器（或网格），画出简单图形绕某点旋转特定角度后的图形。

## Common Pitfalls
- 找错旋转角：这是旋转中最普遍的错误。学生常把图形中两条相交线段的夹角误认为是旋转角。实际上，旋转角必须是**同一点旋转前后的位置与旋转中心连线形成的夹角**（例如 $\angle AOA'$）。
- 旋转中心定位错误：有时旋转中心不在图形上，而在图形外，学生在脑海中想象旋转时会偏离中心。
- 忽略方向：题目说“旋转 $90^\circ$”，学生默认是顺时针，画完后发现答案是逆时针。当题目未明确说明时，通常需要分情况讨论。

## Parent Guidance
- 表盘教学法：家里最好的旋转教具就是带指针的钟表。让孩子观察时针从 $12$ 走到 $3$，旋转中心在哪里（指针转轴），方向是什么（顺时针），角度是多少（$90^\circ$）。
- “找原配”思维训练：给出一张旋转前后的对比图，让孩子拿着彩笔，把原图的顶点 A 和旋转后的顶点 A' 连到旋转中心 O 上。这能极其深刻地帮孩子理解“$\angle AOA'$ 才是真正的旋转角”，并在图上直观看到 $OA = OA'$。

## Linked Mistake Concepts
- 旋转与中心对称混淆：很多学生在学习了这节后，会觉得把图形转一下就是对称。需要强调，旋转可以转任何角度（如 $30^\circ, 45^\circ$），而中心对称特指旋转恰好是 $180^\circ$ 的特殊情况。

## Examples
```text
Question:
如图（想象），\triangle ABC 绕点 C 逆时针旋转 40^\circ 得到 \triangle A'B'C。
(1) 旋转中心是哪一点？旋转角是多少度？
(2) 若 \angle ACA' = 40^\circ，那么 \angle BCB' 等于多少度？
(3) 如果 AC = 5cm，那么 A'C 等于多少？

Solution:
(1) 根据题意，图形绕点 C 转动，所以旋转中心是 点 C。
题目明确说明旋转了 40^\circ，所以旋转角是 40^\circ。

(2) 根据旋转的性质：“任意一对对应点与旋转中心的连线所成的角都等于旋转角”。
点 A 旋转到点 A'，点 B 旋转到点 B'，它们的旋转中心都是点 C。
因此 \angle ACA' 和 \angle BCB' 都是旋转角。
所以 \angle BCB' = \angle ACA' = 40^\circ。

(3) 根据旋转的性质：“对应点到旋转中心的距离相等”。
点 A 的对应点是 A'，旋转中心是点 C。
所以线段 A'C 等于原图形中的线段 AC。
因为 AC = 5cm，所以 A'C = 5cm。
```
