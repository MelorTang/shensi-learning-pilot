# 二次函数的图像和性质

## Core Idea
二次函数的图像是一条抛物线（Parabola）。它具有开口方向、对称轴、顶点等基本特征。理解这些特征及其与函数解析式系数的关系，是掌握二次函数的关键。

## Learning Steps
1. 学习用描点法画出简单的二次函数图像。
2. 观察抛物线的形状，认识抛物线的开口方向、顶点和对称轴。
3. 探讨二次函数在顶点处的最大值或最小值。
4. 归纳二次函数的增减性（在对称轴两侧的单调性变化）。

## Common Pitfalls
- 画图时取点过少或取点不具有代表性，导致无法画出平滑的抛物线。
- 混淆对称轴的表示方法，常写成一个数值（如 2）而不是一条直线方程（如 $x=2$）。

## Parent Guidance
鼓励孩子多动手画图，通过直观的图像来理解抽象的性质。可以使用图灵计算器或几何画板等数字工具，让孩子动态观察函数图像的变化。

## Linked Mistake Concepts
- 顶点与最值混淆：顶点是图像上的一个点 $(x, y)$，而最值是顶点的纵坐标 $y$ 的值。
- 增减性描述不完整：必须指明在哪个区间（对称轴的左侧还是右侧），函数值随自变量如何变化。

## Examples
```text
题目：
下列关于抛物线 y = x^2 的说法中，错误的是（）
A. 开口向上
B. 对称轴是 y 轴
C. 顶点坐标是 (0,0)
D. 当 x > 0 时，y 随 x 的增大而减小

解答：
对于抛物线 y = x^2：
A正确：因为二次项系数 a=1 > 0，所以开口向上。
B正确：由于没有一次项和常数项，它的对称轴是 y 轴（直线 x=0）。
C正确：当 x=0 时，y 取最小值 0，顶点为 (0,0)。
D错误：因为对称轴为 x=0，开口向上，所以在 x > 0 的区域（对称轴右侧），y 随 x 的增大而增大。

故选 D。
```
