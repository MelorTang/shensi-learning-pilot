# 三角形的内角和

## Core Idea
三角形内角和定理（三个内角的和等于180°）及其推论（外角性质）是计算角度、证明角关系的重要基石。外角性质（三角形的一个外角等于与它不相邻的两个内角的和）常常能起到化繁为简的作用，是解决复杂角度问题的捷径。

## Learning Steps
1. **探索并证明内角和定理**：通过拼图或画平行线辅助线，严谨证明内角和为180°。
2. **应用内角和求角度**：在已知两个角或各角比例关系时，求算三角形的各个内角。
3. **理解三角形的外角**：明确外角是由一条边的延长线与另一条相邻边组成的角。
4. **掌握并应用外角性质**：外角等于不相邻两内角和，外角大于任何一个不相邻的内角。

## Common Pitfalls
- **外角找错**：错把对顶角或者其他不符合定义的角当成三角形的外角。
- **舍近求远**：在明明可以使用外角定理一步得出的情况下，非要绕道内角和定理，增加了计算量和出错率。
- **辅助线添加错误**：在证明内角和定理时，不知道过顶点作对边的平行线。

## Parent Guidance
可以让孩子动手剪一个三角形，把三个角撕下来拼在一起，直观看到它们拼成了一个平角（180°）。在孩子做求角度的题目时，如果孩子做了很多步才算出来，家长可以启发：“能不能找找外角？用外角定理会不会更快？”以此培养孩子多角度思考的能力。

## Linked Mistake Concepts
- 平行线的性质（内角和定理的证明强烈依赖于平行线性质）
- 比例的计算（常常出现“三个内角之比为1:2:3”这类结合比例的考法）

## Examples
题目：在△ABC中，∠A = 50°，∠B = 60°，求∠C的度数及与∠C相邻的外角的度数。
```text
解：
根据三角形内角和定理：
∠A + ∠B + ∠C = 180°
50° + 60° + ∠C = 180°
∠C = 180° - 110° = 70°。

设与∠C相邻的外角为 ∠ACD。
方法一：利用平角定义
∠ACD = 180° - ∠ACB = 180° - 70° = 110°。

方法二：利用三角形外角性质
∠ACD = ∠A + ∠B = 50° + 60° = 110°。
答：∠C的度数为70°，与∠C相邻的外角的度数为110°。
```
