# 有理数的大小比较

## Core Idea

比较两个有理数的大小，本质上是在数轴上比较它们的位置：右边的点表示的数总比左边的点表示的数大。代数法则上，正数大于一切负数和0；两个负数比较，绝对值大的反而小。

## Learning Steps

- 判断两数的符号：正数 > 0 > 负数。
- 同号数比较：如果是两个正数，绝对值大的数大；如果是两个负数，绝对值大的数**反而小**。
- 异号数比较：正数永远大于负数。
- 借助数轴：在数轴上把点画出来，靠右的数字更大。

## Parent Guidance
在辅导两个负数比较大小时，可以通过“欠钱”的例子帮助孩子理解：“欠3.5元和欠2元，哪种情况你的资产更多（或者说更穷）？”欠得越多，表示财富越少。或者直接让孩子画数轴，建立直观的数形结合思维。

## Common Pitfalls

- **两个负数比较出错**：例如认为 -5 > -3，因为 5 > 3，忘记了负数比较大小时绝对值大的反而小。
- **带分数的负数比较**：在比较类似 -1/2 和 -1/3 时，通分后比较绝对值出错，或者没有反转大小关系。

## Linked Mistake Concepts
- 负数大小比较错误
- 分数比较错误

## Examples

### Example 1

题目：
比较大小：-3.5 和 -2

解法：

```text
方法一（法则法）：因为 |-3.5| = 3.5，|-2| = 2。3.5 > 2，所以两个负数比较，绝对值大的反而小，即 -3.5 < -2。
方法二（数轴法）：在数轴上，-3.5 位于 -2 的左侧，所以 -3.5 < -2。
```
