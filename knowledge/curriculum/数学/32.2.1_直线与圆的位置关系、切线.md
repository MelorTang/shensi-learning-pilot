# 直线与圆的位置关系、切线

## Core Idea
直线和圆的位置关系有三种，取决于圆心到直线的距离 $d$ 与圆的半径 $r$ 之间的大小关系：
1. **相交**：直线和圆有两个交点，即 $d < r$。这条直线称为圆的**割线**。
2. **相切**：直线和圆有唯一一个交点，即 $d = r$。这条直线称为圆的**切线**，交点称为**切点**。
3. **相离**：直线和圆没有交点，即 $d > r$。

代数判定法（$d$ 与 $r$ 的比较）是解决直线与圆位置关系最根本的方法，将几何问题转化为代数不等式。

## Learning Steps
1. **直观感知**：画一个圆，用直尺在圆上移动，观察直线与圆交点个数的变化。
2. **定量分析**：学习如何通过计算圆心到直线的距离 $d$ 来判断位置关系。
3. **理解切线定义**：明确相切时，交点“唯一”。
4. **综合应用**：结合一元一次方程、不等式和勾股定理求解未知数范围。

## Common Pitfalls
- 圆心到直线的距离 $d$ 计算错误。经常有学生将点到直线的距离直接用某条线段的长代替，而没有验证是否为**垂线段**。
- “相切”与“相交”混淆，题目中说“直线与圆有公共点”，很多同学只想到相交（$d<r$），忘了相切（$d=r$）也是有公共点的（所以是 $d \le r$）。

## Parent Guidance
- 拿一枚硬币和一根直尺，在桌面上平移直尺，让孩子说出直尺什么时候与硬币相交、相切、相离。
- 提醒孩子“公共点”是一个统称，如果有“至少一个公共点”，一定要包含相交和相切两种情况。
- 如果孩子解题遇到困难，问他：“圆心在哪里？这条直线的垂线段怎么画？距离算出来没有？”

## Linked Mistake Concepts
- 点到直线的距离定义不清晰（必须是垂线段）
- 一元一次不等式的解法错误
- 忽视特殊情况导致漏解

## Examples

**Example 1**
已知 $\odot O$ 的半径为 $4\text{ cm}$，点 $O$ 到直线 $l$ 的距离为 $3\text{ cm}$，判断直线 $l$ 与 $\odot O$ 的位置关系。

**Solution**
```text
已知圆的半径 r = 4 cm，圆心到直线的距离 d = 3 cm。
因为 d = 3 < 4 = r，
所以直线 l 与 ⊙O 相交。
```

**Example 2**
已知 $\odot O$ 的圆心到直线 $l$ 的距离 $d = 5$，若直线 $l$ 与 $\odot O$ 有公共点，求 $\odot O$ 的半径 $r$ 的取值范围。

**Solution**
```text
直线 l 与 ⊙O 有公共点，意味着直线与圆可能相交，也可能相切。
当直线与圆相交时，d < r；
当直线与圆相切时，d = r。
因此，直线与圆有公共点的条件是 d ≤ r。
因为 d = 5，
所以 5 ≤ r，即 r ≥ 5。
```
