# 用列举法求事件概率

## Core Idea
当随机试验的结果比较复杂，或者包含多个步骤（如连掷两次硬币、从两个袋子里各摸一个球）时，要准确无误地找出所有等可能的结果总数 $n$ 和符合条件的结果数 $m$，直接口算极易出错。
此时我们需要使用**列举法**。常用的列举法有两种：
1. **列表法**：适用于涉及**两步试验**或两个元素的问题。通过建立一个二维表格，行代表第一个元素的可能结果，列代表第二个元素的可能结果，表格中的每一个格子就是一种组合。
2. **画树状图法**：适用于涉及**两步及以上试验**（如三步或更多）的问题。从树根开始，一步一步展开分支，最后数一数树叶的数量，就是总的可能结果数 $n$。

**核心提示**：使用列举法求概率的隐含前提依然是各个结果必须是**等可能**的。如果是“放回”和“不放回”抽样，树状图和表格的画法会产生显著差异。

## Learning Steps
1. **判断试验步数**：审题，看是几步试验。两步可以用列表法或树状图；三步及以上通常只能用树状图。
2. **区分“放回”与“不放回”**：
   - **放回（或独立重复）**：例如掷两次骰子，第一次的结果不影响第二次。表格的对角线可以存在。
   - **不放回**：例如从三个球中先摸出一个不放回，再摸出一个。表格的对角线（两次摸出同一个球）要划掉；树状图第二步的分支会减少。
3. **正确列出所有结果**：保证**不重不漏**。计算总格子数或树叶数得到 $n$。
4. **寻找符合条件的结果**：在表或树状图中，圈出符合题目要求的结果，数出个数 $m$。
5. **计算概率**：应用公式 $P(A) = \frac{m}{n}$，并化简。

## Common Pitfalls
- **混淆“放回”与“不放回”**：这是丢分的重灾区。题目说“同时抽出两张卡片”，等价于“抽一张不放回，再抽第二张”，此时绝不可能抽出两张一样的卡片。
- **只画图不写结论**：画了完美的树状图，但在最后一步忘了写出 $n$ 和 $m$ 是多少，也没有代入公式得出最终概率，导致扣分。
- **无序与有序混淆**：例如抛两枚硬币，“一正一反”包含“（正，反）”和“（反，正）”两种情况，学生容易漏掉一种。

## Parent Guidance
- 和孩子一起玩“石头剪刀布”，让孩子用列表法画出两人出拳的所有可能（3×3=9种），算一算平局的概率（3/9=1/3），以及某一方获胜的概率。这种互动能极大地提升孩子列表的熟练度。
- 重点检查孩子的画图习惯。树状图一定要有“开始”节点，每一层的分支要对齐；列表法的左上角要写清楚行列代表什么。

## Linked Mistake Concepts
- 对“有放回”与“无放回”的逻辑辨析不清
- 排列组合中“先后顺序”带来的差异理解不到位
- 分数运算基础薄弱

## Examples

**Example 1**
有两个不透明的袋子，A袋中有标注为数字 1, 2 的两个球，B袋中有标注为数字 3, 4, 5 的三个球。从 A 袋随机摸出一个球，再从 B 袋随机摸出一个球，求两个球上的数字之和为奇数的概率。

**Solution**
```text
这是一个典型的两步试验，且不受放回与否的影响（因为从两个不同的袋子摸），适合用列表法。

| A袋 \ B袋 |   3    |   4    |   5    |
| :-------: | :----: | :----: | :----: |
|     1     | (1, 3) | (1, 4) | (1, 5) |
|     2     | (2, 3) | (2, 4) | (2, 5) |

由表可知，共有 6 种等可能的结果，即 n = 6。
两数之和分别为：
1+3=4, 1+4=5(奇), 1+5=6
2+3=5(奇), 2+4=6, 2+5=7(奇)
和为奇数的结果有：(1, 4), (2, 3), (2, 5)，共 3 种，即 m = 3。
所以，P(和为奇数) = 3 / 6 = 1/2。
```

**Example 2**
小明、小红和小华三个人通过抽签决定谁去打扫卫生。签筒里有三根签，其中一根有标记，抽中者去打扫。小明先抽（不放回），然后小红抽。画树状图求小红抽中的概率。

**Solution**
```text
设这三根签为：中、不1、不2。
画树状图：
            开始
          /   |    \
小明：  中   不1   不2
       / \   / \   / \
小红：不1 不2 中 不2 中 不1
(注意：因为是不放回，小红不可能抽到和小明一样的签)

由树状图可知，所有等可能的结果共有 6 种。
其中小红抽到“中”的结果有 2 种（分别是：小明不1-小红中，小明不2-小红中）。
所以 P(小红抽中) = 2 / 6 = 1/3。
(这个例子也证明了抽签的先后顺序不影响公平性)
```
