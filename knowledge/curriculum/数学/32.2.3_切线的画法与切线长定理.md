# 切线的画法与切线长定理

## Core Idea
**切线长定理**：从圆外一点可以引圆的两条切线，它们的切线长相等，这一点和圆心的连线平分这两条切线的夹角。
**核心结论**：如果从圆外一点 $P$ 引圆 $O$ 的两条切线 $PA, PB$（$A, B$为切点），则有：
1. $PA = PB$（切线长相等）。
2. $\angle APO = \angle BPO$（或者说 $PO$ 平分 $\angle APB$）。
3. 直线 $PO$ 垂直平分弦 $AB$。

**切线的画法**：即如何证明一条直线是切线。通常有两种方法：
- **有点连圆心，证垂直**：如果已知直线与圆的交点，连接交点与圆心，证明连线与该直线垂直。
- **无点作垂线，证等径**：如果不知道直线与圆的交点，从圆心向直线作垂线段，证明垂线段的长度等于圆的半径。

## Learning Steps
1. **理解定理证明**：通过构造全等三角形（$\triangle PAO \cong \triangle PBO$，利用 HL 判定），自己推导出切线长定理的结论。
2. **掌握切线判定法**：熟记证明切线的两句口诀：“有点连圆心，证垂直”、“无点作垂线，证等径”。
3. **识别基本图形**：在复杂的几何图形中，敏锐地识别出“圆外一点引两条切线”的模型，直接应用定理。
4. **综合应用**：结合切线性质、等腰三角形、垂径定理进行多步证明和计算。

## Common Pitfalls
- **混淆“切线”与“切线长”**：切线是直线，没有长度；切线长是指从圆外一点到切点之间的线段的长度。
- **判定切线方法选择错误**：当交点已知时去作垂线段算距离往往很难，当交点未知时却盲目连线，导致无从下手。
- **忽视连带结论**：只记得 $PA = PB$，却忘了连心线 $PO$ 是角平分线以及 $PO \perp AB$。

## Parent Guidance
- 提醒孩子背诵切线判定的口诀：“有点连圆心，证垂直；无点作垂线，证等径”。每次做切线证明题前，先问问自己属于哪一种情况。
- 画一个圆外一点引两条切线的图形，让孩子把能发现的相等角、相等线段、垂直关系全标出来，这有助于培养孩子对几何图形的直觉。

## Linked Mistake Concepts
- 直角三角形全等的判定（HL）掌握不牢
- 切线判定方法与性质方法的混用
- 角平分线与垂直平分线的性质应用错误

## Examples

**Example 1**
如图，从 $\odot O$ 外一点 $P$ 引 $\odot O$ 的两条切线 $PA$、$PB$，$A$、$B$ 为切点。若 $\angle APB = 60^\circ$，$PA = 8$，求弦 $AB$ 的长。

**Solution**
```text
因为 PA、PB 分别是 ⊙O 的切线，
根据切线长定理，PA = PB。
又因为 ∠APB = 60°，
所以 △PAB 是等边三角形（有一个角是 60° 的等腰三角形是等边三角形）。
因此，AB = PA = 8。
```

**Example 2**
已知 $AB$ 是 $\odot O$ 的直径，点 $C$ 在 $\odot O$ 上，过点 $C$ 作 $\odot O$ 的切线 $l$。连结 $OC$，求证：$OC \perp l$。这是否证明了 $l$ 是切线？

**Solution**
```text
原题逻辑有误，如果已知 l 是切线，那么“OC ⊥ l”是应用切线的性质，而不是在证明切线。
正确的切线证明题型应该是：
已知 C 在 ⊙O 上，过 C 点作直线 l，且满足 ∠OCA = ...（或者有其他条件推导出 OC ⊥ l），
因为点 C 在 ⊙O 上（有点），
连结 OC（连圆心），
通过条件证明 OC ⊥ l（证垂直），
所以直线 l 是 ⊙O 的切线。
```
