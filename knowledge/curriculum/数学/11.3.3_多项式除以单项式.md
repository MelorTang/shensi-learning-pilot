# 多项式除以单项式

## Core Idea
多项式除以单项式的法则：先把这个多项式的每一项分别除以单项式，再把所得的商相加。
用字母表示为：$(a + b + c) \div m = a \div m + b \div m + c \div m$ （$m \neq 0$）。
这其实是乘法分配律的逆向应用，即分数加法的拆分。核心思想是将“多项式除以单项式”转化为多个“单项式除以单项式”的问题。

## Learning Steps
1. 明确各项及符号：圈出多项式中的每一项，一定要连同它前面的正负号一起圈出来。
2. 逐项分配：将多项式的每一项分别除以那个单项式，写出相加减的算式。
3. 转化为单项式除法：利用前面学的单项式除以单项式的法则，依次求出各项的商。
4. 合并结果：将各个商的代数式连接起来，注意正负号的判定。

## Common Pitfalls
- 漏项：多项式中有某一项恰好和除式完全相同时，相除的商为 $1$。学生常把这个 $1$ 漏写，导致商的项数比被除式少了一项。
- 符号错误：在多项式的后几项除以除式时，忘记带上该项原有的负号，或者除式本身是负数时，正负号变换出错。
- 局部除法：只将多项式的第一项除以单项式，后面的项直接照抄。这是对除法分配性质理解不到位。

## Parent Guidance
- 数项数防漏：给孩子立个规矩——“被除式有几项，商就必须有几项”。做完题目后，让孩子数一数原多项式是三项式，结果是不是也有三项。如果变两项了，大概率是漏了商为 $1$ 的那一项。
- “切西瓜”比喻：多项式除以单项式，就像把一个由好几块切片组成的大西瓜（多项式）平均分给一群人（单项式），每一块切片都要参与平分，不能只分第一块剩下的就不管了。

## Linked Mistake Concepts
- 逆向思维缺失：不习惯将 $(a+b)/c$ 拆分为 $a/c + b/c$，特别是在后续遇到分式化简时，多项式除以单项式是通分加减的逆过程。
- 与提公因式混淆：在化简时，有些学生试图先对多项式提取公因式再约分。这本身是个好方法，但如果提取不全反而容易出错，不如严格按分配法则逐步除稳妥。

## Examples
```text
Question:
计算：
(1) (6x^3 y^2 - 3x^2 y^3 + 9x^2 y^2) \div (3x^2 y^2)
(2) (15a^3 b - 5a^2 b) \div (-5a^2 b)

Solution:
(1) (6x^3 y^2 - 3x^2 y^3 + 9x^2 y^2) \div (3x^2 y^2)
  // 将多项式的每一项分别除以单项式
  = 6x^3 y^2 \div 3x^2 y^2 - 3x^2 y^3 \div 3x^2 y^2 + 9x^2 y^2 \div 3x^2 y^2
  = 2x - y + 3
  // 注意：最后一项相除得到 3，千万不能漏掉。结果项数（3项）与原多项式项数一致。

(2) (15a^3 b - 5a^2 b) \div (-5a^2 b)
  // 分别相除，注意带上项的符号和除式的负号
  = [15a^3 b \div (-5a^2 b)] + [-5a^2 b \div (-5a^2 b)]
  = -3a + 1
  // 注意：第二项 -5a^2 b 除以 -5a^2 b 的结果是 +1，不能写成 0 或省略不写。
```
