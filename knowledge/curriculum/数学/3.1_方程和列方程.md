# 方程和列方程

## Core Idea (核心概念)
方程是含有未知数的等式。列方程是指将实际问题中的数量关系，运用字母表示未知数，并根据问题中的相等关系列出含有未知数的等式。方程是代数学习的基础，是从算术思维向代数思维跨越的重要一步。

## Learning Steps (学习步骤)
1. **理解等式与方程的区别**：认识到所有的方程都是等式，但等式不一定都是方程。方程必须同时满足两个条件：一是等式，二是含有未知数。
2. **寻找等量关系**：在实际应用问题中，学会找出题目中隐藏的“相等”关系，例如“总价 = 单价 × 数量”或“甲的年龄 = 乙的年龄 + 5”。
3. **设定未知数并列式**：用字母（如 $x$, $y$ 等）表示问题中的未知量，并将找到的等量关系用含有字母的等式表达出来。

## Common Pitfalls (常见易错点)
- 容易将不含未知数的等式（如 $3+5=8$）误认为方程。
- 在列方程时，左右两边的单位不统一，导致列出的方程不符合实际意义。
- 漏掉题目中的隐藏条件，无法准确找到等量关系。

## Parent Guidance
家长在辅导孩子学习这一节时，可以多从生活中的实际场景出发。例如带孩子去超市购物时，可以提问：“我们买了3个苹果和2个梨，一共花了15元。已知梨是3元一个，你能用一个带有 $x$ 的式子来表示苹果的单价并列出等式吗？”通过生活中的具体例子，帮助孩子从具体的数字计算平移到抽象的字母表示，克服对未知数的恐惧。鼓励孩子大声读出等量关系式，然后再动笔写出方程。

## Linked Mistake Concepts
- 【算术思维固化】习惯于从已知量逐步推出未知量，而不习惯将未知量作为已知条件参与运算。
- 【代数式的混淆】将代数式（如 $2x+3$）与方程（如 $2x+3=0$）混淆，不明白方程必须有“等号”。

## Examples (例题)

**例题 1**：下列式子中，哪些是方程？
① $3x - 5 = 7$
② $2a + 3b$
③ $4 + 5 = 9$
④ $y^2 = 0$

```text
① 是方程，因为它既是等式，又含有未知数 x。
② 不是方程，因为它没有等号，只是一个代数式。
③ 不是方程，因为它虽然是等式，但不含未知数。
④ 是方程，既是等式，又含有未知数 y。
所以，方程有 ① 和 ④。
```

**例题 2**：一本笔记本的单价是 $x$ 元，小明买了5本，付给售货员20元，找回了3元。请根据题意列出方程。

```text
寻找等量关系：付出的总钱数 - 买笔记本花掉的钱数 = 找回的钱数。
买5本笔记本花费的钱数为 5x 元。
列出方程：20 - 5x = 3
或者也可以根据：买笔记本花掉的钱数 + 找回的钱数 = 付出的总钱数
列出方程：5x + 3 = 20
```
