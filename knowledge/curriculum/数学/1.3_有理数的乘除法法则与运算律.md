# 有理数的乘除法法则与运算律

## Core Idea

有理数的乘除法是在小学乘除法基础上加入了“符号的判定”。核心法则是“同号得正，异号得负”，即先确定结果的符号，再把绝对值相乘除。除法同样可以转化为乘法（乘以倒数）。

## Learning Steps

- **乘法法则**：两数相乘，同号得正，异号得负，并把绝对值相乘。任何数与0相乘都得0。
- **多个数相乘的符号**：奇数个负数相乘结果为负，偶数个负数相乘结果为正（注意有0则结果直接为0）。
- **除法法则**：除以一个不等于0的数，等于乘这个数的倒数。然后再按乘法法则进行计算。
- **运算律应用**：乘法交换律、结合律以及乘法对加法的分配律。

## Parent Guidance
在辅导乘除法时，强调“两步走战略”：第一步先看正负号，第二步再算数字。可以让孩子拿两支不同颜色的笔，红笔专门用来先写下最终符号，黑笔再写数字计算，培养严谨的计算习惯。如果涉及分配律，让孩子画出“分配的箭头”防止漏乘。

## Common Pitfalls

- **乘法分配律漏乘**：在使用分配律 a(b+c) = ab+ac 时，常出现错写成 ab+c 的情况。
- **乘法分配律符号错误**：如果括号外是一个负数，如 -3(2 - 4)，展开时忘记变号，写成 -6 - 12。
- **除法转化为乘法时倒数写错**：比如带分数除法时没有先化为假分数就直接求倒数，或者倒数连带符号一起弄错（正负号不能丢）。

## Linked Mistake Concepts
- 多有理数乘法符号判断错误
- 分配律漏乘
- 带分数求倒数错误

## Examples

### Example 1

题目：
计算：(-1/2) * (-4) * (-3)

解法：

```text
第一步，判定符号：共有3个负号（奇数个），所以最终结果符号为“-”。
第二步，绝对值相乘：(1/2) * 4 * 3 = 6。
综合：最终结果为 -6。
```
