# 尺规作图

## Core Idea
尺规作图是几何学中古老而经典的领域。本章的核心在于明确“尺”（没有刻度的直尺，用于画直线）与“规”（圆规，用于画圆或弧并截取等长线段）的功能限制，熟练掌握基本作图方法。通过线段垂直平分线和角平分线的尺规作图，不仅能直观感受到几何图形的对称美，还能将作图过程与几何判定、性质定理深度结合，培养严密的逻辑推理与动手操作能力。

## Learning Steps
1. **认识尺规作图**
   - 理解尺规作图的工具限制：只能使用无刻度的直尺和圆规。
   - 掌握基本的作图语言，如“以点O为圆心，以任意长为半径画弧”、“连接AB并延长”等。

2. **线段的垂直平分线**
   - 学习并掌握线段垂直平分线的性质定理：线段垂直平分线上的点到线段两端点的距离相等。
   - 学习其判定定理：到线段两端点距离相等的点，在这条线段的垂直平分线上。
   - 掌握用尺规作图法作一条已知线段的垂直平分线。

3. **角的平分线**
   - 学习角平分线的性质定理：角平分线上的点到角两边的距离相等。
   - 学习其判定定理：角的内部到角两边距离相等的点在角的平分线上。
   - 掌握用尺规作图法作一个已知角的平分线。

4. **综合与应用**
   - 利用基本作图进行几何图形的构造，如已知三边长作三角形、过直线外一点作垂线等。
   - 将作图与证明结合，学会说明作图方法的合理性与正确性。

## Common Pitfalls
- 作图时为了图面“干净”，擦除了辅助画弧的痕迹，导致老师无法判断是否使用了尺规作图，扣分严重（作图痕迹必须保留）。
- 对线段垂直平分线和角平分线性质理解偏差，例如把“到角两边距离相等”错误理解成连接某点与原点的线段。
- 在描述作图步骤时，语言不够规范严谨，缺乏诸如“交点为P”或“线段MN即为所求”的关键结论性表述。
- 解决实际作图题时不知道如何转化，比如寻找三角形的内心不知是求角平分线的交点，寻找外心不知是求垂直平分线的交点。

## Parent Guidance
- **动手实践**：给孩子准备质量较好的圆规，并在家里一起动手画出各种基础图形。实际动手画图能极大消除孩子对作图题的畏惧感。
- **保留痕迹**：反复向孩子强调“数学考试中作图题绝对不要擦去任何一条弧线”，哪怕画得有些凌乱，这也是评分的关键依据。
- **逻辑转化**：引导孩子思考作图背后的几何原理。画完角平分线后问：“为什么这样画出来的就是角平分线？”让孩子尝试用全等三角形（SSS）的知识去证明作图的正确性。

## Linked Mistake Concepts
- 【定理混淆】经常混淆角平分线和垂直平分线的性质。垂直平分线对应的是到“两个端点”的距离，而角平分线对应的是到“两条边”的距离（且是垂线段）。
- 【工具滥用】在作图中偷偷使用直尺上的刻度去量取线段长度或找中点，而不是依靠圆规截取。
- 【作高线的错误】在要求作某钝角三角形的高时，忘记延长底边，导致弧线无法与边相交。

## Examples

**Example 1: 作线段的垂直平分线**
已知线段 $AB$，用尺规作图作出其垂直平分线。

```text
作法：
1. 分别以点 A、B 为圆心，以大于 (1/2)AB 的长为半径画弧。
2. 两弧相交于 P、Q 两点。
3. 过 P、Q 两点作直线 PQ。
直线 PQ 即为线段 AB 的垂直平分线。
（证明依据：PA=PB，QA=QB，到线段两端点距离相等的点在垂直平分线上，两点确定一条直线）
```

**Example 2: 垂直平分线的性质应用**
在 $\Delta ABC$ 中，$BC=8$， $AB$ 的垂直平分线交 $BC$ 于点 $D$，交 $AB$ 于点 $E$。已知 $\Delta ADC$ 的周长为 $12$，求 $AC$ 的长度。

```text
因为 DE 是线段 AB 的垂直平分线，且交 BC 于点 D，
根据垂直平分线的性质，有 AD = BD。
已知 \Delta ADC 的周长为 12，即：
AD + CD + AC = 12。
将 AD 替换为 BD，得到：
BD + CD + AC = 12。
观察图形发现，BD + CD 正好等于线段 BC 的长度。
所以 BC + AC = 12。
已知 BC = 8，代入上式得：
8 + AC = 12。
解得 AC = 4。
```

**Example 3: 作角的平分线**
已知 $\angle AOB$，用尺规作图作出它的平分线。

```text
作法：
1. 以点 O 为圆心，以任意长为半径画弧，分别交 OA 于点 C，交 OB 于点 D。
2. 分别以点 C、D 为圆心，以大于 (1/2)CD 的长为半径画弧，两弧在 ∠AOB 的内部相交于点 M。
3. 画射线 OM。
射线 OM 即为 ∠AOB 的平分线。
（证明依据：OC=OD，CM=DM，OM=OM，所以 \Delta OCM ≌ \Delta ODM (SSS)，从而 ∠COM=∠DOM）
```
