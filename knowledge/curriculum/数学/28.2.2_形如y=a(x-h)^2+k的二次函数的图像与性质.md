# 形如y=a(x-h)^2+k的二次函数的图像与性质

## Core Idea
二次函数的顶点式 $y=a(x-h)^2+k$ 能够直接反映出抛物线的顶点坐标为 $(h,k)$，对称轴为直线 $x=h$。通过理解图像的平移规律（“左加右减，上加下减”），可以将最基本的 $y=ax^2$ 平移得到顶点式图像。

## Learning Steps
1. 学习由 $y=ax^2$ 向上下平移得到 $y=ax^2+k$。
2. 学习由 $y=ax^2$ 向左右平移得到 $y=a(x-h)^2$。
3. 综合两种平移，得到 $y=a(x-h)^2+k$。
4. 归纳平移法则，并能直接从顶点式中读出抛物线的顶点、对称轴和最值。

## Common Pitfalls
- 平移方向记忆混淆，尤其是对 $x$ 的操作。例如，将 $y=a(x-3)^2$ 误认为是向左平移（其实是向右平移）。
- 顶点坐标的符号提取错误，如将 $y=2(x+1)^2-3$ 的顶点写成 $(1, -3)$，正确应为 $(-1, -3)$。

## Parent Guidance
帮助孩子总结口诀“括号内管左右，左加右减；括号外管上下，上加下减”。可以通过让孩子自己编写平移过程来检验是否真正理解。

## Linked Mistake Concepts
- 对平移实质的误解：平移改变的是图像的位置，而不改变图像的形状和开口方向（即 $a$ 值不变）。
- 在求最值时，忘记写出是在 $x$ 取何值时取得最值。

## Examples
```text
题目：
将抛物线 y = 2x^2 先向左平移 3 个单位，再向上平移 4 个单位，求平移后的抛物线解析式及顶点坐标。

解答：
根据平移规律“左加右减，上加下减”：
向左平移 3 个单位，对 x 进行操作，x 变为 (x + 3)；
向上平移 4 个单位，对整体 y 进行操作，加上 4。
所以平移后的解析式为：y = 2(x + 3)^2 + 4。

从解析式可以看出，它符合顶点式 y = a(x - h)^2 + k，
其中 h = -3, k = 4。
因此顶点坐标为 (-3, 4)。
```
