# 二次函数y=ax^2+bx+c的图像与性质

## Core Idea
一般式 $y=ax^2+bx+c$ ($a \neq 0$) 是二次函数最常用的形式。通过配方法，可以将一般式转化为顶点式，从而求出对称轴 $x = -\frac{b}{2a}$ 和顶点坐标 $(-\frac{b}{2a}, \frac{4ac-b^2}{4a})$。

## Learning Steps
1. 复习一元二次方程的配方法。
2. 将 $y=ax^2+bx+c$ 运用配方法化简为顶点式形式。
3. 推导出对称轴和顶点坐标的公式。
4. 学习如何根据 $a, b, c$ 的符号判断抛物线在坐标系中的大致位置（图像的草图）。

## Common Pitfalls
- 配方过程中容易在提取系数 $a$ 时出错，特别是常数项的处理。
- 对称轴和顶点公式记忆不清，常把符号记错（例如把 $-\frac{b}{2a}$ 记成 $\frac{b}{2a}$）。

## Parent Guidance
引导孩子不要死记硬背顶点公式，而是要熟练掌握配方法的过程。只要会配方，随时可以自己推导出顶点和对称轴。也可以教孩子通过“左同右异”口诀判断对称轴的位置（$a, b$ 同号对称轴在 $y$ 轴左侧，异号在右侧）。

## Linked Mistake Concepts
- $c$ 的几何意义：误以为 $c$ 与顶点有关，实际 $c$ 是抛物线与 $y$ 轴交点的纵坐标，即交点为 $(0, c)$。
- 二次函数与 $x$ 轴交点个数与 $\Delta = b^2-4ac$ 的关系不清晰。

## Examples
```text
题目：
求二次函数 y = -2x^2 + 8x - 5 的对称轴和顶点坐标。

解答：
方法一：配方法
y = -2(x^2 - 4x) - 5
y = -2(x^2 - 4x + 4 - 4) - 5
y = -2[(x - 2)^2 - 4] - 5
y = -2(x - 2)^2 + 8 - 5
y = -2(x - 2)^2 + 3
因此，对称轴为直线 x = 2，顶点坐标为 (2, 3)。

方法二：公式法
a = -2, b = 8, c = -5
对称轴：x = -b/(2a) = -8 / (2 * -2) = 2
将 x = 2 代入原方程求顶点纵坐标：
y = -2(2)^2 + 8(2) - 5 = -8 + 16 - 5 = 3
所以顶点坐标为 (2, 3)。
```
