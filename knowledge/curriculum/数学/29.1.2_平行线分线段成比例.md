# 平行线分线段成比例

## Core Idea
平行线分线段成比例定理指出：三条平行线截两条直线，所得的对应线段成比例。其推论通常应用于三角形中：平行于三角形一边的直线截其他两边（或两边的延长线），所得的对应线段成比例。

## Learning Steps
1. 观察平行线截两条直线的图形，直观感知对应线段的比值相等。
2. 学习并证明平行线分线段成比例定理。
3. 将定理应用到三角形中，得出推论（如 A 型图、X 型图）。
4. 运用该定理及推论计算线段长度或证明比例式。

## Common Pitfalls
- “对应线段”找错：在复杂的交叉图形（如 X 型图）中，未能正确识别哪两条线段是一一对应的。
- 误用比例：在三角形 $ABC$ 中，$DE \parallel BC$，学生常写出 $\frac{AD}{DB} = \frac{DE}{BC}$ 的错误比例式（正确应为 $\frac{AD}{AB} = \frac{DE}{BC}$）。

## Parent Guidance
鼓励孩子用彩色笔在图形中描出成比例的线段，尤其是用相同颜色标记对应的一对线段。这能极大降低视觉上的混淆，提高正确率。

## Linked Mistake Concepts
- 平行与成比例的互逆关系：有时学生会错误地认为只要成比例就一定平行（实际上还需要满足一定的位置关系，如对应点在同侧）。
- 定理推论中的相似性质混入：把截线段成比例与三角形相似混为一谈，导致在证明步骤中逻辑跳跃。

## Examples
```text
题目：
在 △ABC 中，点 D、E 分别在边 AB、AC 上，且 DE // BC。已知 AD = 4，DB = 2，AE = 6，求 EC 的长。

解答：
因为 DE // BC，根据平行线分线段成比例定理的推论：
AD / DB = AE / EC

代入已知数据：
4 / 2 = 6 / EC
2 = 6 / EC

解得：
EC = 6 / 2
EC = 3

所以，EC 的长为 3。
```
