# 投影

## Core Idea
投影是空间物体在光线照射下，在某个平面（通常是地面或墙面）上留下的影子。投影可以分为两类：由点光源发出的光线形成的“中心投影”和由平行光线（如太阳光）形成的“平行投影”。投影是将三维立体图形转化为二维平面图形的重要手段。

## Learning Steps
1. 观察生活中的影子现象（如灯光下的手影、阳光下树的影子），建立投影的直观概念。
2. 学习投影的几个基本要素：光源（或光线方向）、物体、投影面。
3. 了解投影的分类依据（光线是相交于一点还是互相平行），区分中心投影和平行投影。
4. 探索在不同投影方式下，物体的形状、大小与其投影形状、大小的关系。

## Common Pitfalls
- 将投影与截面混淆：投影是从外部将图形压平到一个面上，而截面是用平面切开物体露出的面。
- 认为物体的投影一定与物体本身的形状相似。实际上，如果物体所在平面与投影面不平行，其投影会发生变形（如圆的投影变成椭圆）。

## Parent Guidance
在晚上带孩子散步时，可以让孩子观察自己在路灯下的影子。当靠近路灯和远离路灯时，影子的长度是如何变化的？这能非常直观地帮助孩子理解中心投影的特点。

## Linked Mistake Concepts
- 忽略光源的位置：在画中心投影时，未能将物体的所有关键点与点光源连接，导致影子画错。
- 混淆正投影与平行投影：正投影只是平行投影的一种特殊情况（光线垂直于投影面），很多学生误认为平行投影就是正投影。

## Examples
```text
题目：
下列现象中，属于中心投影的是（）
A. 阳光下人的影子
B. 探照灯照在墙上形成的影子
C. 阳光下旗杆的影子
D. 舞台上追光灯照射演员在地上形成的影子

解答：
平行投影的光线是平行的，通常自然界中的太阳光被视为平行光。
中心投影的光线是从一个点发出的，如灯泡、手电筒、探照灯等。
A 选项是阳光，属于平行投影。
B 选项探照灯是点光源，属于中心投影。
C 选项是阳光，属于平行投影。
D 选项追光灯是点光源，属于中心投影。

题目可能存在多选或表述问题，但标准答案中，灯光产生的影子（如B、D）都是中心投影，而阳光产生的（如A、C）是平行投影。通常此类单选题会给出一个明显是灯光的情境，故选 B（或D）。
```
