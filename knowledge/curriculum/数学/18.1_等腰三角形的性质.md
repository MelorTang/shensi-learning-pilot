# 等腰三角形的性质

## Core Idea
等腰三角形是有两条边相等的三角形。它不仅是一个特殊的三角形，更是一个典型的轴对称图形。其核心性质有两个：一是“等边对等角”（底角相等）；二是“三线合一”（顶角的平分线、底边上的中线、底边上的高互相重合）。这两个性质是解决角相等、线段相等以及垂直问题的高效工具。

## Learning Steps
1. **理解等腰三角形的概念**：识别腰、底边、顶角、底角。
2. **掌握等边对等角**：在同一个三角形中，相等的边所对的角也相等，常用于角度的计算。
3. **深刻理解“三线合一”**：知道只要满足（1）是等腰三角形，（2）有一条“线”（如角平分线），就能直接推出另外两条“线”（如中线、高）的性质。
4. **轴对称性**：认识到等腰三角形是轴对称图形，对称轴是底边上的高（或中线、顶角平分线）所在的直线。

## Common Pitfalls
- **“三线合一”前提遗漏**：在普通三角形中乱用三线合一，或者忘记说明是“顶角”的平分线、“底边”上的高和中线。
- **边或角的不确定性**：当题目说“一个等腰三角形的边长为3和7”，学生常忘记用三边关系检验（3,3,7是不成立的）。当给出“一个角为50°”时，忘记分顶角和底角两种情况讨论。

## Parent Guidance
重点关注分类讨论思想的培养。当题目出现“等腰三角形的一边...”或“等腰三角形的一角...”时，一定要提醒孩子：“是腰还是底？”“是顶角还是底角？”。在辅导“三线合一”时，可以让孩子在纸上画等腰三角形，对折后发现中线、高、角平分线确实是一条折痕。

## Linked Mistake Concepts
- 三角形内角和定理（已知一个角求另外两个角时必用）
- 三角形三边关系（判断分类讨论的边长是否能构成三角形的唯一标准）
- 全等三角形（等腰三角形性质的证明依赖于全等）

## Examples
题目：已知等腰三角形的一个内角为 70°，求它的另外两个内角的度数。
```text
解：
分两种情况讨论：
情况一：若 70° 角为顶角。
设两个底角分别为 x。
根据三角形内角和定理及等边对等角性质：
70° + x + x = 180°
2x = 110°
x = 55°
此时另外两个内角为 55°，55°。

情况二：若 70° 角为底角。
则另一个底角也为 70°。
设顶角为 y。
70° + 70° + y = 180°
140° + y = 180°
y = 40°
此时另外两个内角为 70°，40°。

答：它的另外两个内角的度数分别为 55°、55° 或 70°、40°。
```
