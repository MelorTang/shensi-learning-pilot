# 二次函数的简单应用

## Core Idea
二次函数在现实生活中有广泛的应用，特别是涉及“最大化”、“最小化”的最值问题（如利润最大、面积最大）以及抛物线轨迹问题（如桥梁拱门、投掷运动）。将实际问题转化为二次函数模型是解题的关键。

## Learning Steps
1. 学习阅读实际问题并提取关键数学信息，设定自变量和因变量。
2. 根据数量关系列出二次函数的解析式，并确定自变量的实际取值范围。
3. 利用配方法或顶点公式求出最值。
4. 验证求出的最值是否在自变量的实际取值范围内，并给出实际问题的结论。

## Common Pitfalls
- 忽略自变量的实际取值范围。例如求得顶点横坐标为 $x=5$，但实际条件限制 $x$ 只能取到 4，导致最值取错。
- 列利润函数时，混淆了“单件利润”与“总利润”的关系：总利润 = 单件利润 × 销售量。

## Parent Guidance
带孩子去超市购物时，可以顺便探讨一下商品降价促销与销售量、利润之间的关系。让孩子明白数学模型来源于生活，又服务于生活。

## Linked Mistake Concepts
- 顶点最值与区间最值混淆：如果抛物线的顶点不在自变量允许的范围内，则需要在边界处寻找最大值或最小值。
- 在抛物线形建筑（如拱桥）题目中，建系不当导致解析式非常复杂。建系通常以顶点为原点或对称轴为 $y$ 轴最简单。

## Examples
```text
题目：
某商店经销一种商品，每件进价为 40 元。经市场调查发现，当售价为 50 元时，每个月能卖出 500 件；售价每提高 1 元，每个月就会少卖 10 件。为了实现每月 8000 元的利润，且尽量减少库存，售价应定为多少元？当售价定为多少时，该月利润最大？最大利润是多少？

解答：
设售价提高了 x 元，即售价为 (50 + x) 元。
此时每件的利润为：(50 + x) - 40 = (10 + x) 元。
每月的销售量为：(500 - 10x) 件。

设每月的总利润为 y 元，则：
y = (10 + x)(500 - 10x)
y = 5000 - 100x + 500x - 10x^2
y = -10x^2 + 400x + 5000

1) 求利润为 8000 元的售价：
-10x^2 + 400x + 5000 = 8000
10x^2 - 400x + 3000 = 0
x^2 - 40x + 300 = 0
(x - 10)(x - 30) = 0
解得 x1 = 10, x2 = 30。
为了尽量减少库存，应使销售量最大，即售价应尽量低，所以取 x = 10。
此时售价为 50 + 10 = 60 元。

2) 求最大利润：
对 y = -10x^2 + 400x + 5000 配方：
y = -10(x^2 - 40x) + 5000
y = -10(x - 20)^2 + 4000 + 5000
y = -10(x - 20)^2 + 9000
因为 -10 < 0，所以当 x = 20 时，y 取最大值 9000。
此时售价为 50 + 20 = 70 元。

答：为了减少库存，售价应定为 60 元。当售价定为 70 元时利润最大，最大利润为 9000 元。
```
