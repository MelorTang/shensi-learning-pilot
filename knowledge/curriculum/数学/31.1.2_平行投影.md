# 平行投影

## Core Idea
平行投影是由相互平行的光线（如阳光）照射物体所形成的投影。正投影是平行投影的一种特殊情况，即投射线垂直于投影面。在平行投影下，物体的实际高度与其影子的长度成正比（在同一时刻），这是解决平行投影计算题的核心依据。

## Learning Steps
1. 认识平行投影的特征：所有投射线互相平行。
2. 学习如何画出简单物体在平行光下的投影。
3. 掌握核心性质：在同一时刻、同一地点，不同物体的高度与它们影长的比值是一个常数。
4. 了解正投影的概念，探讨线段、平面图形在正投影下的性质（如线段垂直于投影面时投影为一个点，平行于投影面时投影长度等于原长）。

## Common Pitfalls
- 混淆“同一时刻”的前提条件。如果题目给出的是不同时刻的影长，不能使用高度与影长成正比的结论。
- 在画平行投影时，把光线画得相交（变成了中心投影）。

## Parent Guidance
可以在一个阳光明媚的下午，和孩子一起在户外测量一根筷子的长度及其影长，然后再测量一棵树的影长，利用比例算出树的高度。这就是最经典的平行投影应用，也是泰勒斯测量金字塔高度的方法。

## Linked Mistake Concepts
- 太阳光位置的动态变化：太阳并不是固定在某一个点的光源，而是随着时间移动，因此平行光线的倾斜角在一天中是不断变化的。
- 正投影的特殊性：当平面图形倾斜于投影面时，其正投影的面积会变小（通过乘以 $\cos\theta$），学生对此难以建立空间想象。

## Examples
```text
题目：
在同一时刻的阳光下，身高 1.5 米的小华在地面上的影长为 2 米。同时，测得一棵大树在地面上的影长为 8 米。求这棵大树的高度。

解答：
在同一时刻的阳光（平行投影）下，物体的高度与它的影长成正比。
设大树的高度为 x 米。

根据比例关系：
小华身高 / 小华影长 = 大树高度 / 大树影长
1.5 / 2 = x / 8

解这个比例方程：
2 * x = 1.5 * 8
2x = 12
x = 6

答：这棵大树的高度为 6 米。
```
