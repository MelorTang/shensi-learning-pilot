# 绝对值

## Core Idea

一个数的绝对值，在几何上表示数轴上这个数对应的点到原点的距离。因为是距离，所以绝对值永远是一个非负数（大于或等于0）。

## Learning Steps

- 理解绝对值的几何意义：点到原点的距离。
- 掌握绝对值的代数求法：正数的绝对值是它本身，负数的绝对值是它的相反数，0 的绝对值是 0。
- 学会进行含有绝对值符号的简单代数运算。

## Parent Guidance
绝对值是初中数学的第一个难点。当孩子去绝对值出错时，一定让他回到数轴上想“距离”的概念。提问孩子：“如果一个数的绝对值是5，那么他在数轴的哪边？是有几种可能？”培养孩子分类讨论的思维。

## Common Pitfalls

- **负数的绝对值等于它本身**：错写成 |-5| = -5，忘记了绝对值代表距离，不能是负数。
- **去绝对值符号时漏掉分类讨论**：当遇到 |a| 时，如果没有说明 a 是正是负，直接写 |a| = a 是一种极其常见的思维漏洞。
- **运算顺序错误**：在形如 |-3| + |-2| 的计算中，没有先算绝对值再相加，导致直接加减内部的数字。

## Linked Mistake Concepts
- 绝对值概念不清
- 去绝对值符号分类讨论缺失

## Examples

### Example 1

题目：
已知 |x| = 6，求 x 的值。

解法：

```text
因为绝对值表示到原点的距离为 6，数轴上到原点距离为 6 的点有两个，分别是 6 和 -6。
所以 x = 6 或 x = -6。
```
