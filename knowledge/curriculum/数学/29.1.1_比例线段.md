# 比例线段

## Core Idea
两线段的比是指在同一单位下这两条线段长度的比值。四条线段 $a, b, c, d$ 中，如果 $a:b = c:d$（或 $\frac{a}{b} = \frac{c}{d}$），则称这四条线段成比例。比例的基本性质如交叉相乘、合比性质、等比性质是解决比例线段问题的主要代数工具。

## Learning Steps
1. 掌握线段比的定义，强调长度单位的一致性。
2. 学习四条线段成比例的定义，识别比例的内项、外项、第四比例项。
3. 掌握比例的基本性质：若 $\frac{a}{b} = \frac{c}{d}$，则 $ad = bc$。
4. 学习合比性质和等比性质，并能进行简单的代数变形和计算。

## Common Pitfalls
- 求“第四比例项”时，没有严格按照题干给出的线段顺序。例如求 $a, b, c$ 的第四比例项，正确的比例式是 $\frac{a}{b} = \frac{c}{x}$，学生可能任意排列导致结果错误。
- 在使用等比性质 $\frac{a_1+a_2+\dots+a_n}{b_1+b_2+\dots+b_n} = k$ 时，忽略了分母之和不能为零的前提条件。

## Parent Guidance
通过解方程的方式帮助孩子复习比例的基本性质。可以让孩子自己设定几条线段的长度，尝试找出哪些线段可以组成比例，培养数感。

## Linked Mistake Concepts
- 比例尺的概念：图上距离与实际距离的比，有时学生会把比值颠倒。
- 代数变形与几何意义脱节：在推导线段比例式时，只关注字母运算而忽略了这些字母代表的是正数（线段长度）。

## Examples
```text
题目：
已知 a, b, c, d 是成比例线段，其中 a = 3, b = 4, c = 6。
(1) 求线段 d 的长；
(2) 若 (a+b)/b = (c+x)/d，求 x 的值。

解答：
(1) 因为 a, b, c, d 成比例，所以 a/b = c/d。
即 3/4 = 6/d
解得 3d = 24，d = 8。

(2) 利用合比性质：若 a/b = c/d，则 (a+b)/b = (c+d)/d。
因此，(c+x) 应该等于 (c+d)。
代入已知数值验证：
左边 = (3+4)/4 = 7/4
右边 = (6+x)/8
所以 7/4 = (6+x)/8
解得：56 = 4(6+x)
56 = 24 + 4x
32 = 4x
x = 8
这与合比性质推导出的 x = d = 8 一致。
```
