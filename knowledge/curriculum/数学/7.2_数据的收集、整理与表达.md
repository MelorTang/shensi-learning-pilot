# 数据的收集、整理与表达

## Core Idea
经历收集、整理、描述和分析数据的完整过程，掌握统计表、条形统计图、折线统计图和扇形统计图的特点及绘制方法。

## Learning Steps
1. 了解数据收集的常用方法，包括全面调查（普查）和抽样调查。
2. 学会制作频数统计表，对原始数据进行归类。
3. 掌握条形统计图、折线统计图和扇形统计图的绘制规范。
4. 能从图表中准确读取信息并做出合理推断。

## Common Pitfalls
- 绘制扇形统计图时，圆心角计算错误或未标明百分比。
- 折线统计图的坐标轴刻度设置不均匀。
- 在条形统计图中忘记写标题或坐标轴的单位。

## Parent Guidance
鼓励孩子收集家庭一个月的开支数据并制作统计图表，培养他们用数据说话的习惯。可以一起看新闻中的统计图，讨论图表想要传达的信息是否客观。

## Linked Mistake Concepts
- 不能根据数据特征选择合适的统计图（例如：看趋势用折线图，看比例用扇形图）。
- 混淆频数和频率，忘记频率之和始终为1。

## Examples
题目：
某班有40名学生，其中喜欢篮球的有20人，喜欢足球的有10人，喜欢乒乓球的有10人。若要绘制扇形统计图表示这组数据，求喜欢足球人数所在扇形的圆心角度数。

解答：
```text
喜欢足球的人数占总人数的比例 = 10 / 40 = 25%。
对应扇形的圆心角 = 360° * 25% = 90°。
```
