# 分式的乘除

## Core Idea
分式的乘除法则与小学分数的乘除法则完全一致：
- 乘法法则：两个分式相乘，把分子相乘的积作为积的分子，把分母相乘的积作为积的分母。即 $\frac{A}{B} \cdot \frac{C}{D} = \frac{AC}{BD}$。
- 除法法则：两个分式相除，把除式的分子和分母颠倒位置后，再与被除式相乘。即 $\frac{A}{B} \div \frac{C}{D} = \frac{A}{B} \cdot \frac{D}{C} = \frac{AD}{BC}$。
乘方运算同样适用：分式的乘方，就是把分子、分母分别乘方。即 $(\frac{A}{B})^n = \frac{A^n}{B^n}$。

## Learning Steps
1. 转化除法为乘法：一拿到题目，如果看到除号，第一步永远是把除号变乘号，后面紧跟的分式分子分母上下颠倒。
2. 全面因式分解：对所有的分子和分母进行因式分解，全部转化为连乘积的形式。
3. 上下约分：在乘法状态下，分子和分母（即使不在同一个分数线上）只要有相同的因式就可以约去。
4. 写出最简结果：把约分后剩余的因式相乘，通常结果也要保持为最简分式或整式（不需要把因式再展开乘成多项式形式，除非题目特别要求）。

## Common Pitfalls
- 颠倒错误：遇到连除或者乘除混合运算，如 $A \div B \div C$，学生容易弄错该翻转谁。法则：紧跟在“$\div$”号后面的那个分式必须翻转，即 $A \cdot \frac{1}{B} \cdot \frac{1}{C}$。
- 未分解就相乘：遇到多项式相乘，有的学生直接用多项式乘法展开得到一个极大的多项式，导致无法再化简。正确做法是**先因式分解，先约分，最后再乘**。
- 负号处理不当：因式分解时产生的负号（例如提公因式或 $a-b$ 变为 $-(b-a)$ 时），没有统一提到分式最前面，导致约分时漏掉或符号混乱。

## Parent Guidance
- “先解剖，再相爱”：告诉孩子，两个庞大的分式相乘，不要直接拥抱（展开乘法），那样会变成怪物。要先把它们“解剖”（因式分解）成最小的零件，看看哪些零件一样，抵消（约分）掉，剩下清清爽爽的再组合。
- 重点盯防“多项式的符号”：若看到 $x-y$ 和 $y-x$ 出现，提醒孩子这是“伪装的同类”，提个负号就能消灭它们。

## Linked Mistake Concepts
- 与分式加减的混淆：分式乘除不需要通分找公分母！部分学生一看到两个分式，条件反射去通分，把简单问题极度复杂化。

## Examples
```text
Question:
计算：
\frac{x^2 - 1}{x^2 - 2x} \div \frac{x^2 + 2x + 1}{x}

Solution:
第一步：将除法转为乘法，后一个分式上下颠倒
原式 = \frac{x^2 - 1}{x^2 - 2x} \cdot \frac{x}{x^2 + 2x + 1}

第二步：对所有多项式进行因式分解
x^2 - 1 = (x+1)(x-1)       // 平方差
x^2 - 2x = x(x-2)          // 提公因式
x^2 + 2x + 1 = (x+1)^2     // 完全平方

代入原式：
= \frac{(x+1)(x-1)}{x(x-2)} \cdot \frac{x}{(x+1)^2}

第三步：约分（约去公因式 x 和 (x+1)）
分子剩下：(x-1)
分母剩下：(x-2)(x+1)

第四步：得出结果
= \frac{x-1}{(x-2)(x+1)}
// 结果保留因式相乘形式即可，无需展开分母。
```
