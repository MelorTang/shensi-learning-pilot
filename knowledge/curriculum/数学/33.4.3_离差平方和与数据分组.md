# 离差平方和与数据分组

## Core Idea
在实际统计中，如果数据量非常大，并且已经整理成了**频数分布表**，我们要如何计算这些分组数据的方差？
这就需要用到**加权方差（分组数据的方差公式）**。
设数据分成了 $k$ 组，各组的组中值分别为 $x_1, x_2, \cdots, x_k$，对应的频数分别为 $f_1, f_2, \cdots, f_k$，总数 $n = f_1+f_2+\cdots+f_k$。
那么分组数据的方差公式为：
$$s^2 = \frac{f_1(x_1-\bar{x})^2 + f_2(x_2-\bar{x})^2 + \cdots + f_k(x_k-\bar{x})^2}{n}$$

在这个公式中，**频数 $f_i$ 起到了“权”的作用**。计算的核心依然是算离差平方和，只不过相同的离差要乘以它出现的次数。

## Learning Steps
1. **找组中值**：由于分组后的数据不再是精确数字，我们需要用每组的中间值（上限和下限的平均数）来代表这一组的数据。
2. **算加权平均数**：利用组中值和频数，计算出总体的加权平均数 $\bar{x}$。
3. **算加权离差平方和**：计算每个组中值与平均数的差的平方，**必须**再乘以该组的频数。
4. **求平均得到方差**：将所有加权离差平方和相加后，除以总频数 $n$。

## Common Pitfalls
- **忘记乘频数**：算完组中值与平均数的离差平方后，直接相加，忘记了每一组里有好多个人（频数），导致最终结果极度偏小。
- **除错分母**：最后除以的应该是总人数（频数的总和 $n$），而不是组数 $k$。
- **取错组中值**：没有采用上下限的平均值，而是随便取了组内的某个值。

## Parent Guidance
- 这部分内容的计算量非常大，极容易出错。提醒孩子在草稿纸上画一个规范的五列大表：“组中值 | 频数 | 乘积(求均值用) | 离差平方 | 离差平方×频数”。一步一步填表，能极大地降低出错率。
- 让孩子明白，之所以要用组中值，是因为原始数据已经丢失了，组中值是我们能做出的最合理的“估算”。

## Linked Mistake Concepts
- 加权平均数计算错误
- 频数与组数概念的混淆
- 复杂的四则混合运算顺序和粗心失误

## Examples

**Example 1**
某班 20 名学生的年龄分布如下：
14岁：5人
15岁：10人
16岁：5人
求这 20 名学生年龄的方差。

**Solution**
```text
因为已经给出了具体数值，这里这里的年龄就是 x，人数就是频数 f。
第一步：求加权平均数
平均数 x̄ = (14*5 + 15*10 + 16*5) / 20 = (70 + 150 + 80) / 20 = 300 / 20 = 15。

第二步：求加权离差平方和的均值（即方差）
方差 s² = [5*(14-15)² + 10*(15-15)² + 5*(16-15)²] / 20
         = [5*1 + 10*0 + 5*1] / 20
         = 10 / 20
         = 0.5。
答：这20名学生年龄的方差为 0.5。
```
