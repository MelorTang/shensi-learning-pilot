# 一次式的加减与去括号

## Core Idea

一次式的加减，本质上就是“合并同类项”。合并的法则是：把同类项的系数相加减，字母和字母的指数保持不变。去括号是化简的先决条件。

## Learning Steps

- **去括号法则**：
  1. 括号前面是“+”号，去掉括号和它前面的“+”号，括号里各项的符号都不改变。
  2. 括号前面是“-”号，去掉括号和它前面的“-”号，括号里各项的符号都要改变（正变负，负变正）。
- **找同类项**：带着符号把同类项找出来。
- **合并同类项**：只对系数进行加减运算，字母部分照抄。

## Parent Guidance
“去括号，看符号”是口诀。括号前是负号时，一定要盯紧孩子是不是**每一项**都变了号，很多人只变第一项，忘了后面的项。合并同类项时，告诉孩子：“3个苹果加2个苹果等于5个苹果，苹果还是苹果，不要变成平方了！”

## Common Pitfalls

- **去负号括号时漏变号**：如 $-(x - 3)$ 展开写成了 $-x - 3$，正确应为 $-x + 3$。
- **合并时把指数相加**：如 $2x + 3x$ 错算成 $5x^2$。加法不改变字母的次数！
- **带分数系数计算错误**：涉及到含有分数的系数合并时，通分计算出错。

## Linked Mistake Concepts
- 去括号变号错误
- 合并同类项字母指数改变

## Examples

### Example 1

题目：
化简：$(3x - 5) - (2x - 4)$

解法：

```text
第一步，去括号：
原式 = 3x - 5 - 2x + 4  （注意：括号前是负号，里面各项全变号）

第二步，合并同类项：
= (3x - 2x) + (-5 + 4)
= x - 1
```
