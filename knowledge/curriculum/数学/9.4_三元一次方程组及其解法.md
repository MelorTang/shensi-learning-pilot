# 三元一次方程组及其解法

## Core Idea
了解三元一次方程组的概念，进一步深化“消元”思想，将三元化为二元，再化为一元来逐步求解。

## Learning Steps
1. 认识三元一次方程组：含有三个未知数，并且含有未知数的项的次数都是1的方程组。
2. 观察方程特点，决定消去哪个未知数最简便。
3. 通过代入或加减消元，将三元一次方程组转化为二元一次方程组。
4. 求解二元一次方程组，再依次回代求出三个未知数的值。

## Common Pitfalls
- 消元时思路不明确：第一次由①和②消去了x，第二次由②和③却消去了y，导致得到的新方程组仍然包含三个未知数，无法继续求解。
- 回代时出错，或者回代顺序混乱。

## Parent Guidance
提醒孩子做题时保持耐心，过程书写要条理清晰，标注好①、②、③。建议在草稿纸上写明“消去谁”，确保每次消去的都是同一个未知数。可以通过做题来锻炼孩子的心细程度。

## Linked Mistake Concepts
- 错误地认为三元一次方程组一定要先消去特定的未知数（如x），而不善于观察系数特点寻找最容易消去的目标（如系数为1或相反数的项）。
- 等式两边乘系数时漏项，产生连锁反应导致最终全错。

## Examples
题目：
解三元一次方程组：
{ x + y + z = 6      ①
{ 2x - y + z = 5     ②
{ 3x + y - z = 4     ③

解答：
```text
观察发现，y 的系数有 1 和 -1，容易消去。
① + ② 得：
3x + 2z = 11         ④

② + ③ 得：
5x = 9               
解得 x = 1.8         （其实直接解出了x）

将 x = 1.8 代入 ④：
3(1.8) + 2z = 11
5.4 + 2z = 11
2z = 5.6
解得 z = 2.8

将 x = 1.8，z = 2.8 代入 ①：
1.8 + y + 2.8 = 6
4.6 + y = 6
解得 y = 1.4

所以方程组的解为：
{ x = 1.8
{ y = 1.4
{ z = 2.8
```
