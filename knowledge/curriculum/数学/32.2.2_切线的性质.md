# 切线的性质

## Core Idea
**切线的性质定理**：圆的切线垂直于经过切点的半径。
如果一条直线是圆的切线，那么连接圆心和切点的半径必定垂直于这条切线。
**推论 1**：经过圆心且垂直于切线的直线必经过切点。
**推论 2**：经过切点且垂直于切线的直线必经过圆心。

在解题时，“见切线，连切点”是黄金辅助线法则，连完之后立刻得到一个垂直关系（直角），从而可以运用直角三角形的各种性质（勾股定理、三角函数等）求解。

## Learning Steps
1. **理解定理**：从切线的定义（只有一个公共点）出发，理解为什么距离最近的线段（半径）一定是垂线段。
2. **掌握辅助线作法**：遇到题中有“直线是圆的切线”这个已知条件，第一时间作出辅助线——连接圆心和切点，并标上垂直符号。
3. **结合直角三角形计算**：有了垂直关系，通常会形成直角三角形，结合已知边长和角度，运用勾股定理或三角函数进行计算。

## Common Pitfalls
- **辅助线作错**：不知道连哪里，常常将切点连接到圆周上其他点，而不是连接圆心。
- **倒果为因**：把性质当成了判定。性质是“已知是切线，得出垂直”，而在证明切线时，是要“证明垂直，得出是切线”。
- 忘记切线和半径的垂直关系，导致无法构造直角三角形。

## Parent Guidance
- 问孩子一句话：“题目里说这条线是切线，你要做的第一件事是什么？” 答案必须是：“连接圆心和切点，画上直角符号。”
- 如果孩子解题卡壳，提醒孩子检查是否遗漏了切线带来的隐藏条件（即 $90^\circ$ 角）。
- 在草稿纸上画一个圆，画一条切线，然后让孩子随意连接圆心和切线上的点，问哪个线段最短？自然得出只有连接切点的那条半径是最短的（即垂线段）。

## Linked Mistake Concepts
- 勾股定理及三角函数公式记忆不牢
- 逻辑推理顺序颠倒（判定与性质混淆）
- 证明题格式不规范

## Examples

**Example 1**
如图，直线 $AB$ 切 $\odot O$ 于点 $P$，连结 $OP$。若 $\angle A = 30^\circ$，$OA = 6$，求 $OP$ 的长。（此处假设 $A$ 在直线 $l$ 上，$O$ 到 $A$ 距离为6）

**Solution**
```text
因为直线 AB 是 ⊙O 的切线，P 为切点，
根据切线的性质，OP ⊥ AB。
在 Rt△OPA 中，∠OPA = 90°。
又因为 ∠A = 30°，
根据直角三角形中，30°角所对的直角边等于斜边的一半，
所以 OP = 1/2 * OA = 1/2 * 6 = 3。
答：OP 的长为 3。
```

**Example 2**
判断题：垂直于切线的直线一定经过圆心。

**Solution**
```text
错误。
必须是“经过切点且垂直于切线的直线”才一定经过圆心。如果随便在切线上找一点作垂线，不一定经过圆心。
```
