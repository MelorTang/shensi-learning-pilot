# 全等三角形的概念与性质

## Core Idea
全等三角形是指能够完全重合的两个三角形。全等的“核心”在于“重合”，重合意味着它们的大小和形状完全相同。全等三角形的性质非常直接：对应边相等，对应角相等。这是整个初中平面几何证明线段相等和角相等的最重要工具。

## Learning Steps
1. **理解全等图形和全等三角形的概念**：直观感受全等图形是可以通过平移、旋转、翻折重合的图形。
2. **掌握表示方法**：学习用“≌”符号表示全等，并规定将对应顶点的字母写在对应位置上（如△ABC ≌ △DEF）。
3. **识别对应元素**：在两个全等三角形中，重合的边叫对应边，重合的角叫对应角。
4. **掌握全等三角形的性质**：全等三角形的对应边相等，对应角相等，这也是后续做题证明的最常用依据。

## Common Pitfalls
- **对应顶点书写不对应**：在写“△ABC ≌ △DEF”时，随意调换字母顺序，导致对应关系混乱。
- **找错对应边和对应角**：在经过翻折或旋转的图形中，仅凭肉眼直觉找对应元素，而不是通过已知条件的边长或角度大小来严谨判断（如最长边对最长边，最大角对最大角）。

## Parent Guidance
在辅导时，可以利用两块完全相同的三角板，通过平移、旋转、翻转等动作，让孩子直观找出哪条边对应哪条边，哪个角对应哪个角。重点检查孩子的书写规范，看“≌”前后的字母是否严格按对应关系排列。

## Linked Mistake Concepts
- 轴对称、平移和旋转（全等三角形常作为这三种图形变换的背景）
- 等腰三角形（在等腰三角形的证明中大量运用全等）

## Examples
题目：已知 △ABC ≌ △DEF，且 A和D，B和E 是对应顶点。若 AB = 5cm，BC = 6cm，∠A = 65°，求 DE 的长度及对应角。
```text
解：
因为 △ABC ≌ △DEF，且 A和D，B和E，C和F 是对应顶点，
所以根据全等三角形对应边相等：
DE = AB = 5cm。
根据全等三角形对应角相等：
∠D = ∠A = 65°。
答：DE的长度为5cm，对应角∠D为65°。
```
